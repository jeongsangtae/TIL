# 오늘 느낀 점

- 그룹 채팅방 내의 사용자 목록에서 사용자 클릭 시에 사용자 프로필 툴팁이 열리는데, 이 때 다른 사용자를 클릭하면 이전 사용자 프로필 툴팁과 새롭게 클릭한 사용자 프로필 툴팁이 함께 열리는 구조로 구성이 되어있어서 해당 내용을 수정해 이전 사용자 프로필 툴팁이 열린 상태에서 다른 사용자를 클릭하면 이전 사용자 프로필 툴팁은 닫히고 새롭게 클릭한 사용자 프로필 툴팁만 열리도록 구성을 해주었다. 사용자 프로필 툴팁 위치가 채팅 목록 위에 보여지긴 하지만, 생각한 것과 다르게 조금 붕뜬 느낌이 있어서 추가로 위치조정이 필요하며 패널 내의 사용자 목록에서 사용자 클릭 시에 사용자 프로필 툴팁이 보여지도록 구성을 해볼 생각이다. 이 구성들을 먼저 한 뒤 그 다음 내부 CSS 추가적인 기능을 구성하는 방향으로 생각중이며 중간에 더 필요한 내용이나 우선으로 수정해야 할 내용이 생기면 해당 내용을 먼저 수정해 처리해줄 생각이다.
