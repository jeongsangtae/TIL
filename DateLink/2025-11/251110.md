# 오늘 느낀 점

- 사용자가 로그인하면 해당 사용자 정보와 온라인 유무가 그룹 채팅방에 참여한 온라인 상태의 사용자들에게 전달되고, 해당 정보를 통해 그룹 채팅방 내의 사용자 목록 온라인 유무가 실시간으로 변경되도록 구성을 한 내용은 문제없이 작동하지만, 불필요한 정보가 전달되는 부분이 있어서 이 부분을 추가로 수정할 생각이다. 그리고 로그인했을 때 그룹 채팅방 내의 사용자 목록이 실시간 반영되는건 해결을 했기 때문에, 반대로 로그아웃했을 때 그룹 채팅방 내의 사용자 목록이 실시간 반영되도록 구성을 해야 하는데, 이 내용은 이전에 구성한 온라인 친구 목록 내용 또는 로그인했을 때 그룹 채팅방 내의 사용자 목록을 실시간 반영하는 내용을 참고해 구성하면 해결할 수 있을 것 같다는 생각이 들어서 해당 내용들을 살펴보고 약간 수정해 구성할 생각이다.
