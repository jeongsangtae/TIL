# 오늘 느낀 점

- 그룹 채팅방 패널 내의 사용자 목록에서 오프라인 사용자는 옅게 보여지도록 구성해 온라인 사용자와 구분해주었고, 사용자 프로필 수정 폼 내의 모드 전환 버튼과 아바타 이미지 관련 내용을 수정하면서 UI가 이전보다 자연스러워지긴 했지만 조금 부족한 느낌이 들어서 추가로 수정할 생각이다. 우선 초기화 버튼을 따로 빼는 것은 UI상 보기 좋지 않다는 생각이 들어서 입력창 내에 아이콘으로 보이도록 수정할 생각이고, 아바타 이미지 미리보기 내용에서 이미지 URL이 없을 경우 아이콘이나 다른 형태의 내용이 보여져 이미지 미리보는 공간이라는 것을 알 수 있도록 구성해줄 생각이다. 여기에 추가로 로그인 시에 그룹 채팅방 내의 사용자 목록에서 실시간 반영되는 내용에 대한 로직을 추가할까 생각중인데, 굳이 필요한가 라는 생각도 들고 친구가 아닌 사용자 로그인 유무를 아는 것이 맞는지도 의문이 들어서, 조금만 고민해보고 확실하게 결정해 처리할 생각이다.
