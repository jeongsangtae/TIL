# 오늘 느낀 점

- 다이렉트 채팅방과 그룹 채팅방 관련된 내용에 구성되어 사용하지 않는 아바타 관련 내용을 모두 삭제하고, CSS 내용에서는 아바타 관련 내용을 모두 주석 처리했는데 현재까지 큰 문제없이 이전과 동일하게 보여지는 것을 확인했고, 프론트엔드 내의 JSX 코드 내용에서 길었던 아바타 관련 내용이 아바타 컴포넌트로 인해 많이 코드가 줄었다고 생각이 된다. 그리고 그룹 채팅방 내의 사용자 목록 또는 패널에서 오프라인 상태의 사용자 닉네임만 불투명도가 적용되고 아바타는 적용되지 않는 문제가 있었는데, 그룹 채팅방 사용자 그리고 패널 컴포넌트에서 확장 클래스로 오프라인 관련 내용을 전달해 적용시키면서 해당 문제도 해결해주었다. 아바타 컴포넌트를 재사용하면서 구성한 확장 클래스 내용에 다시 한 번 확인을 해봐야 하며, CSS 파일에서 구성된 내용 순서나 겹치는 부분을 확인해 수정이 필요한 부분은 추가로 수정해줄 생각이다.
