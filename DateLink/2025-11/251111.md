# 오늘 느낀 점

- 로그인 시에 로그인한 사용자 정보를 전달하고 있는데 이 정보가 전달만 되고 사용되지 않기 때문에 해당 내용을 수정해 사용자 \_id만 전달하도록 수정하고, 오프라인 시에 그룹 채팅방 내의 사용자 목록이 실시간 반영되어 온라인에서 오프라인 목록으로 옮겨지도록 내용을 추가해주었다. 이 내용들 마지막으로 한 번 테스트해 문제가 없는지 확인을 해봐야 하며, 이전에 수정한 채팅 관련 내용도 다시 한 번 확인해 문제가 없는지 확인을 해볼 생각이다. 그리고 이제 할 일 목록을 다시 확인해 하나씩 정리해줄 생각인데, 이미 처리해놓고 정리하지 않은 내용이 있을 수 있기 때문에 전체적으로 살펴보고 하나씩 정리해줄 생각이다. 마지막으로 여러 내용에 주석을 추가해 어떤 내용인지 명확하게 해주는 쪽이 나중을 위해 좋다고 생각되므로 여러 함수, 변수 등 큰 내용은 흐름에 맞게 주석을 추가할 생각이다.
