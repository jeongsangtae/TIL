# 오늘 느낀 점

- 사용자 프로필 내용이 툴팁처럼 보이도록 CSS를 구성하고, 사용자 프로필에서 아바타 컴포넌트를 추가해 닉네임과 아바타 관련 내용 그리고 온라인 유무가 보여지도록 구성했는데 내용들이 출력되는 것은 문제가 없지만 사용자 프로필 내용이 채팅 목록 뒤에 숨겨져 짤리는 문제가 있기 때문에 portal을 사용해 채팅 목록보다 앞에 아바타 프로필이 보이도록 구성해볼 생각이다. 그리고 사용자 프로필에 전달하는 내용에 대한 타입 정의가 구성되지 않아서 타입 에러가 발생하고 있고, 사용자 여러명 클릭해 프로필을 여러개 여는 구조를 수정해 하나 열린 경우 다른 내용은 닫히도록 수정해봐야 하며, 사용자 프로필에서 내용 클릭 시에 사용자 프로필 모달이 보여지도록 구성해볼 생각이다. 최우선으로 portal 관련 내용을 추가한 뒤 테스트해볼 생각이며 다른 내용은 이후에 순서대로 하나씩 구성할 생각이다.
