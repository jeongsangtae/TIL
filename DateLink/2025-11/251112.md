# 오늘 느낀 점

- 그룹 채팅방 내의 사용자 목록에서 어떤 사용자가 로그인하거나 로그아웃했을 경우, 기존에 로그인한 사용자는 갑자기 오프라인 목록에 보여지는 문제가 발생해 이와 관련된 내용을 수정해 주었고, 친구 관련 내용과 그룹 채팅방 관련 내용 소켓 이벤트에 주석을 추가해 어떤 내용인지 바로 알 수 있도록 해주었다. 여러 내용을 살펴보며 약간 수정이 필요한 부분을 찾았는데, 아바타 관련된 내용이 여러 컴포넌트에서 사용되고 있는데 이 내용은 거의 비슷한 구조로 구성되어 공통된 부분이 많기 때문에 커스텀 훅 또는 재사용 컴포넌트로 구성해 여러 내용에서 재사용되도록 구성하면 JSX 코드 내용을 많이 줄일 수 있다고 생각되어 수정할 생각이다. 이 외에 할 일 내용을 살펴보고 추가로 수정할 내용이나 정리가 필요한 내용을 찾아 수정해줄 생각이다.
