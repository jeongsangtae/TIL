# 오늘 느낀 점

- 사용자 클릭 시에 열리는 사용자 프로필 관련 내용을 그룹 채팅방 디테일 컴포넌트에서 관리하고 관련 내용을 하위 컴포넌트로 내린 뒤 상태 끌어올리기로 정보를 받아 출력하도록 구성했는데, 해당 내용이 큰 문제없이 작동하는 것은 확인했지만 한 번 열린 사용자 프로필 툴팁은 다시 닫을 수가 없는 구조이기 때문에 해당 문제를 해결해야 하며, 내용을 구성하면서 발생한 타입 에러를 모두 수정하지 않았기 때문에 이 부분 또한 추가로 수정을 해줄 생각이다. 그리고 해당 내용들이 모두 수정되면 이전에 사용하던 내용들은 모두 깔끔하게 정리해야 하고, 사용자 프로필 창이 열리는 위치와 크기 또한 좀 더 세세하게 조정해 일정한 크기와 위치를 가지도록 해야 하며, 내부에 보여지는 내용 또한 아직 출력되지 않는 내용이 있어서 추가해야 하고 내용 클릭 시에 모달창이 열려 좀 더 세부적인 내용이 보여지도록 구성해볼 생각이다.
