# 오늘 느낀 점

- 아바타, 사이드바, 알림, 채팅 관련 내용, 친구 관련 내용에 구성되어 사용하지 않는 아바타 관련 내용을 모두 삭제하고, CSS 내용에서는 아바타 관련 내용을 모두 주석 처리해주었는데, 현재까지 큰 문제없이 잘 적용이 되고 있으며 프론트엔드의 JSX 코드 내용에서 길었던 아바타 관련 내용이 아바타 컴포넌트로 재사용되면서 많이 코드가 줄었다고 생각이 된다. 다이렉트 채팅방과 그룹 채팅방 관련 내용에 주석 처리된 아바타 내용 또한 모두 삭제해 정리를 해야 하며, 여기에 확장 클래스로 오프라인 관련 내용을 전달해 오프라인 상태일 때 닉네임만이 아닌 아바타 또한 불투명한 상태로 보이도록 구성할 생각이다.
