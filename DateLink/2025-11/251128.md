# 오늘 느낀 점

- 사용자 프로필 props와 그룹 채팅방 내의 사용자 프로필 포탈 관련 내용에 관한 타입 정의를 수정해 타입 에러를 해결하고, 사용자 프로필 위치가 좀 더 자연스럽게 보이도록 수정한 결과, 타입 에러가 모두 사라진 것을 확인했으며 사용자 프로필 위치도 이전보다 좀 더 적절한 위치에서 보여지는 것을 확인했다. 그리고 그룹 채팅방 패널 내의 사용자 클릭 시에 사용자 프로필이 보여지도록 구성을 해봤으나 생각한 것과 다르게 사용자 프로필이 출력되지 않아서 추가로 수정해볼 생각이다. 패널 컴포넌트가 사용자 목록과 사용자 컴포넌트 구조와 다른 형태로 구성되어 생각한대로 작동하지 않는 것 같은데, 활성화 관련 로직과 좌표 관련 로직을 더 상위 컴포넌트에 옮겨서 전체적으로 관리되도록 수정해 테스트해보고 안된다면 다른 방법으로 사용자 프로필이 열리도록 구성해볼 생각이다.
