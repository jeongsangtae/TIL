# 오늘 느낀 점

- 아바타 관련 내용을 재사용하기 위해 아바타 컴포넌트를 추가하고, 채팅 목록 컴포넌트나 채팅 컴포넌트에서 아바타 컴포넌트에 props를 통해 데이터를 넘겨 테스트한 결과, 크게 문제없이 잘 적용된 것은 확인했지만 className props로 전달하는 내용이 조금 깔끔하지 못한 것 같아서 이름과 내용을 수정해볼 생각이다. 추가로 여러 컴포넌트에서 사용중인 아바타 관련 내용도 아바타 컴포넌트에 props로 데이터를 넘겨 적용되도록 구성을 해야 하는데, 다른 내용들은 문제가 없지만 약간 다른 CSS 내용이나 className 이름 관련해 추가로 전달할 수 있도록 구성해야 하고, 전체적으로 표준화된 크기를 기본으로 잡고 조금씩 다른 부분은 추가 className을 통해 덮어씌우는 방식 또는 상위 컴포넌트에서 미리 구성해 전달하는 방식이면 깔끔할 것 같다는 생각이 들어서 해당 내용들을 다시 테스트해볼 생각이다.
