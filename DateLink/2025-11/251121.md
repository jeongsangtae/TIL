# 오늘 느낀 점

- 아바타 이미지 URL을 수정할 때 잘못된 이미지 URL을 넣으면 미리보기에서 기본 아바타 색과 닉네임 첫 글자가 보여지는데, 이 내용 그대로 제출하면 미리보기와 똑같이 기본 아바타 색과 닉네임 첫 글자가 보이지 않고 기본 그림 이미지와 조금 깨진 듯한 UI가 보여지는 문제가 발생하기 때문에 해당 내용을 수정해 올바른 이미지 URL이 아닐 경우, 이미지 에러 상태로 만들어 기본 아바타 색과 닉네임 첫 글자가 보여지도록 구성했는데 해당 내용이 잘 적용되어 이제는 문제없는 것을 확인할 수 있었다. 얼마전 다이렉트 채팅방 패널 내의 함께 참여한 그룹 채팅방 또는 함께 아는 친구 목록을 펼치고 시크릿 모드의 다른 계정에서 똑같이 해당 내용을 펼쳤을 때 기존에 열린 목록이 다시 닫히는 문제를 발견했기 때문에 토글 관련 조건부 내용을 다시 확인해 수정해줄 생각이며, 해야 할 일 정리와 다이렉트 채팅방 순서가 변경될 때 부드럽게 변경되는 내용 또는 그룹 채팅방 위치를 자유롭게 변경하는 기능을 추가하는 방향으로 구성해볼 생각이다.
