# 오늘 느낀 점

- 아바타 이미지 URL 입력창 내에 초기화 아이콘이 보이도록 구성하고, 아바타 이미지 URL이 존재하지 않으면 빈 공간이 아닌 아바타 색과 닉네임 첫글자가 보여져 UX를 향상시켜주었는데, 아바타 이미지 URL에서 내용이 잘못되거나 조금의 텍스트에도 아바타 색과 닉네임 첫글자가 아닌 빈 공간이 보이기 때문에 이 내용을 수정해 잘못된 URL일 경우 아바타 색과 닉네임 첫글자가 보이도록 구성해볼 생각이다. 그리고 로그인 시에 그룹 채팅방 내의 사용자 목록도 실시간으로 반영되어 보여지도록 구성할 생각인데, 이 사용자 목록의 온라인 유무를 가져오는 것이 온라인 친구 목록 불러오는 것처럼 유사하게 구성하면 가능할지 아직 확실하지 않아서 테스트를 해봐야 하긴 하겠지만, 그래도 해당 내용을 추가하면 UX가 이전보다 향상되기 때문에 추가해볼 생각이다.
