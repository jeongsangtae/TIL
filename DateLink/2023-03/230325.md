# 오늘 느낀 점

- 시간이 나오는 건 UTC를 추가해서 UTC 기준 시간이 나오게 되어 발생하는 문제였다. 단지 내가 제대로 확인하지않고 사용해서 생긴 문제였다. UTC를 빼준 getMonth, getDate, getFullYear를 사용하면 문제가 없는 것으로 확인하였다. 사소한 실수지만 꼭 제대로 확인하고 사용하는게 중요하다고 생각된다.

- localStorage를 사용해서 데이터를 저장하는 부분에서 문제가 있는데 배열을 따로 만들고 객체형식으로 저장하려고 하는데 계속 저장이 되지않아서 이 부분을 해결해야한다. 계속 무엇이 문제인지 찾아보고 확인해보는데 아직 찾지 못하였다. localStorage에 저장하는 것 보다 일단 배열에 input 데이터가 저장되는 것 부터 확인을 해보고 localStorage에 저장하는 방식으로 진행을 해야 원인을 찾고 해결해야한다.

- 다시 앞으로 돌아가서 확인해보면서 작업을 해봐야겠다고 생각이 되는데 JS 로컬 스토리지 게시판 만드는게 더 어렵다고 다시 한번 생각하게 된다.
