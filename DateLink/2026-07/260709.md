# 오늘 느낀 점

- 프론트엔드에 구성된 그룹 채팅방 관련 여러 파일과 레이아웃, UI 내용을 확인해보며 주석이 없는 내용은 주석을 추가하고 설명이 부족한 부분은 내용을 추가해준 뒤 설명이 맞지 않거나 약간 수정이 필요한 부분은 수정해주면서 로직 블록별로 어떤 역할을 하는지 이전보다 명확하게 알 수 있게 되었다. 그룹 채팅방 생성 또는 수정 시에 제목을 입력하는데 이 때 길이에 따른 에러 메시지 출력하는 조건부를 수정해 변수 하나로 처리될 수 있도록 구성하고 공지사항에서 영어 또는 숫자 포함 시 내용이 길어져도 줄바꿈이 되지 않고 한 줄로 보여지는 문제를 해결하기 위해 CSS를 수정해 처리해주었다. 추가로 또 다른 문제를 발견했는데 그룹 채팅방 패널에서 사용자 클릭 시에 열리는 프로필 내용 하단이 잘리는 문제가 있어서 이 부분은 추가로 확인을 해봐야 하는데 현재 이 내용이 클릭한 사용자 위치에 맞춰서 프로필이 열리는데 하단 제한이 없는 상태로 작동되어 잘리는 문제가 발생하는 것 같기에 이 부분을 수정해 보이는 화면 위로 보여지도록 구성해볼 생각이다.
