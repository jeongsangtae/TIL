# 오늘 느낀 점

- 사용자 관련 파일들을 확인해보며 주석이 필요한 부분은 추가해주었고 여러 조건부 내용이 길게 작성되거나 재사용이 가능하다고 생각한 부분은 정리해주었는데 확실히 이전보다 좀 더 보기 편해진 것 같아서 앞으로도 이런 방식으로 수정해줄 생각이며 그룹 채팅방 패널에서 사용자 클릭 시에 보여지는 프로필 위치 관련 테스트 로직은 주석 처리된 상태로 사용하지 않는데 해당 내용을 삭제하면서 불필요한 주석 처리된 내용이 정리된 것을 확인했다. 주석을 추가하면서 몇 가지 내용도 테스트를 했었는데 사용자 프로필 수정에서 닉네임을 길게 작성해 수정하면 그 닉네임 전체 내용이 그대로 화면에 보여지고 프로필 영역에서 닉네임만 사이드바를 벗어나 보여지거나 그룹 채팅방 내의 사용자 클릭 시 프로필 영역에서도 마찬가지로 닉네임만 길게 보여져 기존 영역을 벗어나는 문제가 발생하고 여러 내용에서 불편한 UI로 보여지고 있기 때문에 이 부분 또한 수정해줄 생각이다.
