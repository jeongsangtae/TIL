# 오늘 느낀 점

- 닉네임 그리고 아바타 색 또는 아바타 이미지를 수정할 경우 모달창이 닫히지 않고 toast 메시지가 출력되도록 구성을 하고 닉네임 길이 제한을 15자에서 10자로 줄여 사용자 프로필 수정창에서만 줄인 형태로 보여지고 계정 정보나 그룹 채팅방 내의 사용자 프로필 또는 사용자 프로필 세부 내용에서는 풀 닉네임 형태로 보여지도록 구성해주었다. 실제 존재하는 이메일을 가지고 인증하는 방식의 내용은 현재 구성을 할 수는 있지만 우선순위가 아니며, 일단 UI 정리와 버그 수정 그리고 주석 추가부터 모두 처리한 뒤 README에 향후 구현 예정 기능으로 적어둔 뒤 배포하고 난 후에 구성하는 것으로 방향을 정했다. 그래서 현재 처리할 내용은 Zustand 내용에 구성된 상태나 액션에 주석을 추가해 로직 블록별로 어떤 역할을 하는 로직인지 구분해야 하며 전체적으로 어긋난 UI가 추가로 있는지 기능 작동이 생각한 것과 다르게 동작하는 것은 없는지 확인을 해볼 예정이다.
