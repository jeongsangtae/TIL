# 오늘 느낀 점

- 친구와 그룹 채팅방 스토어 내용을 확인해 구성된 액션 함수에 주석을 추가하고 설명이 부족한 부분은 추가 그리고 수정이 필요한 주석은 수정해 로직 블록별로 구분이 되도록 해주었고 재사용할 수 있는 내용은 따로 함수로 구성한 뒤 소켓 이벤트에 재사용해 코드를 줄여주었다. README 파일을 확인해봤는데 예전에 구성한 내용은 너무 아쉬운 부분이 많고 거의 목차 수준이라서 수정을 해줄 예정인데 여기에 프로젝트 소개와 기술 스택 그리고 주요 기능과 프로젝트 구조를 구성한 뒤 구현한 기능 상세 내용이나 배운 점, 마지막으로 앞으로 구현할 기능 같은 to do 리스트를 구성하는 것까지 생각중이다. 여기에 추가로 실행 방법까지 추가하면 괜찮을 것 같고, 이번에 README를 구성하면서 이전에 README 구성에 대해 부족했던 것까지 채울 생각인데 최대한 완성도 높게 구성해 프로젝트 깊이를 높일 생각이다.
