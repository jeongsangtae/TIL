# 오늘 느낀 점

- 로그인 관련 fetch 함수를 인증 Zustand에 옮겨 실행되도록 구성하고 로그인한 사용자 테마 정보가 로컬 스토리지 테마 항목을 덮어씌워 로그인 시에 바로 해당 사용자 테마 내용이 적용될 수 있도록 구성했는데 모두 문제없이 반영되어 보여지는 것을 확인했다. 로그인 관련 fetch 함수 위치를 옮기면서 추가로 여러 내용도 수정을 했는데 수정한 내용에서 타입 에러가 발생하는 부분도 있고 주석 처리된 내용도 있기 때문에 해당 내용들을 모두 정리해줄 생각이다. 그리고 회원가입 관련 fetch 함수 또한 인증 Zustand에 옮겨 실행되도록 구성해야 하며 비슷하게 타입 에러가 발생할 수 있기 때문에 해당 문제도 함께 처리해줄 생각이다.
