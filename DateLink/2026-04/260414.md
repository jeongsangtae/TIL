# 오늘 느낀 점

- 테마 모드 변경 시에 변경된 정보를 Zustand 액션 함수를 통해 백엔드에 전달하고 백엔드에서 사용자를 확인한 뒤 해당 사용자 정보에 포함된 테마 정보를 전달받은 테마 정보로 업데이트하도록 구성을 했는데, 테스트해본 결과 문제없이 작동되는 것을 확인했고 가입 시에 테마 항목이 포함되는 것 또한 문제없는 것을 확인했다. 하지만 로그인 시에 해당 정보를 불러오고 그 정보로 로컬 스토리지 테마 항목을 업데이트하는 내용이 아직 구현되지 않아서 다른 계정으로 로그인했을 때 여전히 이전 로컬 스토리지에 저장된 테마 항목이 적용되는 문제가 발생하므로 추가 수정을 해줄 생각이다. 그리고 로그인과 회원가입 관련 fetch 함수가 현재 각 컴포넌트 파일에 구성되어 있는 상태인데, 이 방식이 아닌 Zustand에 구성되어 실행되는 방식으로 수정해주는 것이 좋을 것 같아서 해당 내용 또한 수정해줄 생각이다.
