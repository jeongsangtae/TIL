# 오늘 느낀 점

- 여러색 내용을 추가로 확인해 CSS 공용 변수로 만들고 해당 내용들을 적용해주었는데 모두 문제없이 적용되어 이전과 동일하게 보여지는 것을 확인했고, 추가로 테두리 관련 내용도 비슷한 내용이 여러 곳에서 동일하게 사용되는 것을 확인했기 때문에 해당 내용 또한 CSS 공용 변수로 구성해 적용해줄 생각이다. 그리고 박스 그림자 관련 내용과 트랜지션 관련 내용 또한 통일해 CSS 공용 변수로 구성할 생각이고, z-index 내용도 지저분하지 않고 깔끔하게 정리해줄 생각이다. 이 외에 높이나 너비 관련 내용도 확인을 해봐야 할 것 같은데 이 부분은 저번에 굳이 필요없다고 생각도 했었는데 다시 확인해보니 변수화를 하는 것도 괜찮아보여서 다시 제대로 확인해 변수화를 진행할 예정이다.
