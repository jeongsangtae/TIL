# 오늘 느낀 점

- 친구 관련 내용의 검색창을 수정한 뒤 타입 에러가 발생해서 이와 관련된 내용을 처리해주었고, 이전에 아이콘 관련 CSS 공용 변수를 추가한 것과 별개로 다른 아이콘 내용들 색이 조금씩 다른 상태로 여러 곳에서 사용되는 것을 발견해 CSS 공용 변수로 변수화한 뒤 적용해주고 마찬가지로 선 관련 내용 또한 동일한 색으로 여러 곳에서 사용되고 있어서 CSS 공용 변수로 처리해주었는데 모두 문제없이 적용되어 이전 내용과 동일하게 보여지는 것을 확인했다. 색 관련 내용을 전체적으로 확인해 정리를 따로 해놓은 상태인데 이 내용들 중에 여러 곳에서 사용된 내용도 있고 한 곳에서만 사용된 내용도 있는 것을 확인한 상태인데, 여러 곳에서 사용된 내용은 최대한 변수화 작업을 해주고 있고 한 곳에서만 사용된 내용들은 색을 확인해 기존에 변수화한 색 내용과 유사하면 최대한 변수화한 내용으로 수정하고 만약 전혀 다른 색이면 내부에서 사용되도록 놔둔 상태로 유지중인데 좀 더 확인을 해보고 최대한 변수화하는 방향으로 진행해볼 생각이다.
