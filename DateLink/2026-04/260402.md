# 오늘 느낀 점

- 전체 투명도 조절하는 내용을 다시 확인한 뒤 주석 처리된 내용과 불필요한 타입 정의 내용을 삭제해 타입 에러를 해결해주었으며 텍스트와 전송 그리고 온라인 점과 오프라인 점 관련 CSS 공용 변수를 추가해 적용한 결과, 이전과 동일하게 보여지는 것까지 모두 확인을 해주었는데, 여러 내용을 수정하면서 색 관련 내용에서 흰색이 자주 사용되는 것을 발견하게 되었다. 해당 내용이 CSS 공용 변수에 이미 구성이 되어있기 때문에 전체적으로 다시 확인해 CSS 공용 변수 내용을 적용시켜줘야 하며, 이 외에 색 관련 내용에서 조금씩이지만 중복으로 사용된 내용들을 발견해서 이 내용들 또한 추가로 수정해 최대한 색 같은 내용은 CSS 공용 변수로 처리될 수 있도록 해줄 생각이다.
