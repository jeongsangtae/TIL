# 오늘 느낀 점

- 회원가입 성공 상태를 로그인 컴포넌트에 전달하고 로그인 컴포넌트에서 해당 정보를 가지고 회원가입 성공 메시지를 출력하도록 구성한 뒤 테스트한 결과, 성공 메시지가 출력되긴 하지만 타임아웃이 지나기전에 새로고침하면 다시 성공 메시지가 뜨고 회원가입 페이지로 돌아간 후 다시 로그인 페이지로 돌아오면 회원가입 성공 메시지가 출력되는 문제가 발생해 인증 레이아웃에 toast 라이브러리 관련 내용을 추가하고 회원가입 컴포넌트에서 로그인 페이지로 이동하는 함수를 분리한 뒤 toast로 회원가입 성공이 뜨도록 구성한 결과, 앞에서 발생한 문제들이 모두 해결되어 내가 생각한 의도대로 회원가입 성공 toast 메시지가 출력되는 것을 확인했다. 회원가입 성공 메시지 출력 관련 테스트했던 내용들은 모두 삭제해야 하고 이 외에 여러 네트워크 에러에 대한 toast 내용을 추가해 구성해야 하며 계정 삭제에 대한 로직도 없어서 이와 관련된 내용도 추가할 예정이다.
