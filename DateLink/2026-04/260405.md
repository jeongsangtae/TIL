# 오늘 느낀 점

- 친구 관련 내용에 구성된 온라인, 모두, 대기중 각 내용의 검색창을 분리해 개별적으로 검색어가 입력될 수 있도록 수정하고 초기화 아이콘 또한 개별적으로 작동하도록 구성해 테스트해본 결과, 온라인에서 입력한 내용 따로 모두에서 입력한 내용 따로 각각 보여지는 것을 확인했고 서로 이동해봐도 이전에 입력된 각 내용이 그대로 보여지는 것을 확인했으며 초기화 아이콘 또한 각 내용만 삭제하는 것까지 모두 확인되었는데, 한 가지 고민으로 친구 추가 탭으로 이동하고 돌아왔을 때 내용이 초기화되는 것이 맞는지 고민이 많았으나 결국 각 탭에 검색된 내용이 초기화되지 않는 방향으로 결정했기 때문에 내용이 그대로 유지되는 방향으로 진행할 생각이다. 그룹 채팅방 초대 검색창은 단순하게 모달창이 닫히면 검색창이 초기화되도록 구성해 처리를 해주었는데 큰 문제가 없어보이고, 친구 관련 내용에서 검색창을 분리하는 내용으로 수정하면서 타입 에러가 발생한 것을 확인했는데 해당 내용은 추가 수정이 필요할 것으로 생각이 되며, 여러 색 내용을 전체적으로 모두 확인한 결과 CSS 공용 변수로 처리할만한 내용들이 몇 개 보여서 이 내용들 또한 처리해줄 생각이다.
