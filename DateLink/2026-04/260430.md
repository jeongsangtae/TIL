# 오늘 느낀 점

- 사용자 설정과 사용자 프로필 수정 폼에서 사용중인 모달 데이터 대신 사용자 정보로 처리하도록 수정하고 사용자 설정 내의 계정 설정 영역에 닉네임, 사용자명, 이메일, 비밀번호 변경 관련 내용이 보여지도록 구성했는데 아직 비밀번호 변경과 관련된 로직을 제대로 구성하지 못해서 이와 관련된 내용을 추가해 테스트해볼 생각이다. 그리고 비밀번호 변경이 현재는 계정 설정에서 바로 보여지는 상태인데 이걸 수정해 비밀번호 변경 버튼으로 구성하고 클릭 시에 추가 모달이 보여져 비밀번호 변경을 하도록 구성할 것인지 아니면 현재 구조처럼 계정 설정에 들어가면 바로 비밀번호 변경이 가능하도록 할 지 정해야 하는데, 개인적으로는 버튼 클릭 시에 모달이 열리고 그 다음 비밀번호를 변경하도록 하는 것이 좋아보이는데 두 방법 모두 테스트해보고 좀 더 깔끔한 쪽으로 구성해줄 생각이다.
