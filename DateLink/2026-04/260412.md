# 오늘 느낀 점

- 박스 그림자와 트랜지션 내용을 확인해 변수화할 수 있는 내용들은 변수화를 해주었고 여러 CSS 파일을 확인하며 변수화 가능한 내용은 변수화한 내용으로 수정하고 약간 애매한 내용은 추가 수정한 뒤 변수화한 내용을 적용해 확인한 결과 이전과 동일하게 보여져 문제가 없는 것을 확인했다. 높이와 너비 내용도 확인해 변수화를 하려고 했지만 해당 내용들은 무조건 공용 변수로 빼는 건 좋지 못하다고 생각하기 때문에 해당 값이 바뀌면 전체 레이아웃이 영향을 받는 내용만 변수화해줄 생각이다. 그리고 사용자 프로필과 사용자 프로필 세부 내용에서 구성된 아바타와 온라인 점 관련 내용을 아바타 컴포넌트로 재사용 가능한 지 확인이 필요하고, 놓친 알림을 따로 확인할 수 있는 알림 아이콘 같은 내용도 추가가 필요해보이고, 테스트를 진행하면서 다른 계정으로 로그인하면 브라우저에 저장된 테마 모드 내용이 그대로 적용되어 유지되고 있는데 이런 구조가 맞는지 의문이 들어서 해당 내용을 계정별로 테마 모드를 정보를 따로 저장해 로그인하면 서버에서 데이터를 가져와 브라우저에 저장하고 테마 모드 변경 시에 브라우저에 해당 데이터를 저장하면서 동시에 서버에 저장해 나중에 다시 로그인했을 때 저장된 테마 모드를 불러와 적용하도록 하는 구조를 테스트해볼 생각이다.
