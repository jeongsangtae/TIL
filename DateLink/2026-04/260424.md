# 오늘 느낀 점

- 인증, 채팅, 다이렉트 채팅, 소켓 관련 Zustand에 구성된 액션 함수에서 에러 처리된 부분이 alert로 처리되고 있는데 해당 내용을 수정해 toast로 처리되도록 구성하고 로그인과 회원가입 컴포넌트 내용을 추가로 확인해 주석 처리되어 사용하지 않는 내용을 모두 삭제해 정리해주었다. 친구, 그룹 채팅 관련 Zustand를 확인해 액션 함수에 구성된 에러 처리 내용을 alert가 아닌 toast로 처리되도록 수정해야 하며, 테스트중 회원 탈퇴 기능이 없는 것을 확인했기 때문에 이와 관련된 내용도 추가할 예정이며 사용자 프로필 관련 내용에서 사용중인 아바타와 온라인 점 관련 내용이 재사용 컴포넌트가 아닌 내부에 직접 구성되어 사용되고 있기 때문에 해당 내용 또한 수정해줄 생각이다.
