# 오늘 느낀 점

- 테두리 색과 zIndex 내용을 확인해 변수화할 수 있는 내용들은 변수화를 해주었고 해당 내용들을 여러 CSS 파일에 적용한 결과 이전과 동일하게 보여져 문제없는 것을 모두 확인했다. 여러 내용을 확인하고 테스트하면서 몇 가지 추가하거나 보완할 부분을 발견했는데, 사용자 프로필과 사용자 프로필 세부 내용에 구성된 아바타와 온라인 점 내용이 아바타 컴포넌트로 재사용되고 있지 않은 것을 발견했는데 해당 내용들이 재사용될 수 있는지 확인이 필요하며 재사용할 수 있다면 재사용하는 방향으로 수정해줄 생각이고, 참여한 그룹 채팅방이나 다이렉트 채팅방 등 여러 내용에서 알림이 전달될 때 해당 알림을 놓치고 보지 못할 경우 알림이 따로 로그처럼 저장되어 볼 수 있거나 알림 아이콘을 추가해 해당 아이콘에 커서를 올리면 전달받은 알림 관련 내용들이 툴팁 형식으로 나열되어 보여지며 확인할 수 있는 구조로 구성하는 것을 생각해봤는데 해당 기능은 있으면 도움이 될 거 같아서 추가하는 방향으로 생각중이다. 그리고 박스 그림자, 트랜지션, 높이, 너비 같은 내용도 추가로 확인해 변수화가 필요한지 확인한 다음 필요한 내용들만 CSS 공용 변수로 구성해 변수화를 진행해줄 생각이다.
