# 오늘 느낀 점

- 텍스트 색 관련 내용은 바로 적용하고 성공과 성공 호버색 그리고 강한 위험과 강한 위험 호버색, 아이콘 색은 CSS 공용 변수에 내용을 추가한 뒤 적용해주었는데 모두 문제없이 적용되어 이전과 동일하게 보여지는 것을 확인했다. 그리고 성공 호버색은 이전에 좀 더 진한 초록색으로 구성이 되어있었는데 이번에 CSS 공용 변수를 구성하면서 기존보다 조금 연한 초록색으로 구성해 성공색보다 약간만 달라지도록 수정해주었다. 추가로 CSS 공용 변수를 적용하면서 테스트중 약간 문제있는 내용들을 발견했는데 그룹 채팅방에 초대할 때 검색하는 내용과 친구 관련 내용에서 검색할 때 이전에 내용을 입력하고 다른 그룹 채팅방으로 이동하거나 다른 온라인이나 모두 같은 내용으로 이동했을 때 검색창이 초기화되지 않고 그대로 유지되어 있는 것을 발견해서 이와 관련된 내용을 수정해 다른 내용으로 이동 시에 초기화되도록 구성해줄 생각이다.
