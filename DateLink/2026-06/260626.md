# 오늘 느낀 점

- Zustand 스토어 내용들을 모두 확인해 콘솔 로그가 남아있는 부분은 전부 삭제하고 주석 처리되어 사용하지 않는 내용도 모두 삭제 그리고 추후에 다시 구성될 내용은 주석을 남겨 설명을 추가해주었다. 백엔드 내용도 약간 정리를 했는데 아직 하나의 파일만 정리한 상태라서 전체적으로 백엔드 관련 모든 파일을 확인해 정리해줄 생각이며 정리하면서 관련된 내용에서 불필요한 내용도 있다면 함께 삭제할 생각이다. 백엔드까지 마무리가 된다면 전체적으로 내용을 살펴보며 간단하게 주석을 추가해 어떤 로직인지 설명하는 내용을 추가할 예정인데, 너무 세세하게 적는 것 대신 간략하게 어떤 역할인지만 알 수 있도록 주석을 남겨놓을 생각이다.
