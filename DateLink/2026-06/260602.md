# 오늘 느낀 점

- 사용자 프로필 제목과 계정 제목 위치가 중앙에 보여지고 있는데 이 부분이 뭔가 통일성도 떨어지고 조금 구식 같은 느낌이 들어서 왼쪽에 보여지도록 수정하고, 사용자 비밀번호 수정 관련 CSS 수정 그리고 사용자 삭제 관련 CSS는 추가해주었는데 전체적으로 이전보다 나아지긴 했지만 약간 간격이나 크기 같은 여러 미세조정이 필요한 부분들이 많아서 이 부분에 대해 추가 수정할 예정이다. 전체적으로 UI가 좀 더 트렌드있고 깔끔한 느낌으로 보여지도록 하고 싶은데 이런 감각적인 부분이 부족한지 쉽지 않아서 여러 웹 사이트나 어플을 참고해야 할 것 같고 빠르게 CSS 정리를 한 뒤 알림 관련 내용까지 마무리해서 이 프로젝트를 마무리할 생각이다.
