# 오늘 느낀 점

- 알림 버튼 클릭 시에 드롭다운이 열리고 다시 알림 버튼을 클릭해야만 드롭다운이 닫히는 구조였지만 현재는 드롭다운 외부 클릭 시에 닫히도록 수정해 알림 버튼으로만 드롭다운을 닫지 않아도 될 수 있도록 구성을 해주었고, 알림 메시지가 존재하지 않을 경우 알림 제목 부분과 밑줄 그리고 빈 공간이 보여졌는데 해당 내용도 수정해 알림이 없다는 텍스트가 보여져 UI도 이상하지 않고 이전보다 훨씬 자연스러워진 것을 확인했다. 알림이 없을 때 보여지는 문구 내용 관련 CSS를 다시 확인해 추가 수정해야 하고 알림 관련 CSS에 구성된 내용들을 확인해 변수화가 필요한 부분은 변수화해주어야 하며 우측 하단에 보여지는 알림 메시지와 알림 버튼을 통해 보여지는 알림 목록이 동시에 보여질 때 해당 내용 중에서 어떤 내용이 우선순위로 보여지는지 확인이 필요한데 해당 내용은 확인한 뒤 순서를 수정해줄 생각이다. 그리고 사용자 프로필 관련 내용에서도 변수화하지 않은 CSS 내용들이 있는 것 같아서 이 부분 또한 확인해 수정해줄 생각이다.
