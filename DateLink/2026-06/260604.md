# 오늘 느낀 점

- 사용자 설정 관련 내용과 알림 관련 내용을 앞으로 어떤식으로 구성할 지 간단하게 정리해주었는데 이 내용에서 빠진 부분이나 부족한 내용은 구성하면서 더 추가할 예정이다. 사용자 설정 관련 UI 구성은 빠르게 마무리하고 알림 관련 내용을 구성해야 하는데 이 내용에서 알림 버튼이 어디에 위치하면 좋을지 좀 고민해봤는데 테마 버튼이 있는 영역에서 추가하는 것이 제일 좋다는 생각이 되기 때문에 해당 위치에 추가할 생각이며, 알림 메시지가 많이 쌓이게 되면 아래 화면을 가리는 문제가 발생하기 때문에 이 부분은 적당한 길이로 구성하고 알림 메시지가 많아질 경우 스크롤이 생성되어 스크롤로 이동되어 보여지도록 구성할 예정이다.
