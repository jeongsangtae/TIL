# 오늘 느낀 점

- 알림 목록이 열려있을 때 우측 하단에 보여지는 알림 토스트 내용이 보이지 않도록 조건부를 추가하고 공용 변수가 저장된 CSS 파일에 폰트 두께 관련 변수화한 내용을 추가해 업데이트하고 알림 관련 CSS 내용에 구성된 항목 중에서 변수화한 내용으로 수정이 필요한 내용들은 모두 수정한 결과 이전과 동일하게 보여지는 것을 확인했다. 이제 사용자 프로필 또는 사용자 프로필 세부 내용에 구성된 아바타와 온라인 점을 확인해 아바타 컴포넌트로 재사용하도록 수정해야 하고 전체적으로 내용을 확인해 사용하지 않는 내용 또는 주석 처리되어 필요없는 내용들을 모두 삭제해야 하고 각 내용에 주석을 추가해 어떤 역할을 하는지 정리가 필요한 상황인데 여기에 추가로 더 필요한 내용이 있는지 확인을 해봐야 하며 해당 내용들이 모두 마무리되면 배포한 뒤 일렉트론을 사용해 파일 형식으로 만들고 투명도 조절 내용을 테스트해볼 생각이다.
