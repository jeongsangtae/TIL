# 오늘 느낀 점

- 공용 변수에 폰트 관련 내용을 추가한 뒤 사용자 프로필, 계정 설정, 비밀번호 수정, 계정 삭제 관련 CSS 내용에 공용 변수 내용을 추가해 수정한 결과 이전과 동일하게 보여져 문제가 없는 것을 확인했다. 사용자 설정 화면에서 프로필 내용이 활성화될 경우 프로필 버튼이 호버된 효과가 보여지도록 구성해야 하고 반대로 계정 설정 내용이 활성화된 경우 계정 버튼이 호버된 효과가 보여지도록 내용을 추가해야 하는데, 이 기능은 활성화 관련 내용을 추가하면 나름 빠르게 수정이 가능할 것 같아서 적용할 생각이며, 사용자 설정에서 버튼 호버나 활성화되었을 때 테마별로 색이 적용되는데 이 내용이 적절한 지 확인이 필요하며 더 적절한 색 조합이 있다면 그 내용으로 수정할 생각이다. 그리고 알림 관련 내용도 간단하게 추가해 UI보다 기능을 먼저 구성해볼 생각이다.
