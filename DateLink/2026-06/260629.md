# 오늘 느낀 점

- CSS 내용을 전체적으로 한 번 더 확인해 여기서 추후에 필요하다고 생각되는 부분을 제외한 모든 내용은 정리해주었으며 백엔드 내용에서 DB와 JWT 인증 내용을 확인해 코드만 보고 바로 이해할 수 있는 부분에 작성된 주석은 삭제하고 불필요하게 긴 설명으로 작성된 주석은 간략하게 설명하도록 수정해주었으며 부연 설명이 필요한 주석은 약간 내용을 추가해 좀 더 도움이 되는 주석으로 보이도록 수정해주었다. 그리고 사용자 라우터에서 토큰 관련 내용을 확인해 만료 시간이 구현 단계 시간이 아닌 실제 서비스 시에 유효한 시간으로 변경해주었는데, 테스트용 만료 시간은 삭제하지 않고 추후에 다시 테스트가 필요할 때 바로 사용할 수 있도록 남겨두었다. 백엔드의 채팅 관련 라우터부터 사용자 관련 라우터까지 모두 확인해 주석이 필요한 부분은 추가하고 수정이 필요한 부분은 수정해줄 생각이며 너무 긴 내용이 아닌 로직 블록마다 간략하게 필요한 설명만 추가해줄 생각이다.
