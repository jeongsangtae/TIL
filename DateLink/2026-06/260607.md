# 오늘 느낀 점

- 사용자 설정 내의 계정에 구성된 계정 정보와 위험 영역 간의 간격을 조절하고 위험 영역 섹션 제목 색을 수정해 강조 효과를 주고 사용자 비밀번호 수정 내의 레이블과 에러 메시지 내용을 수정해 좀 더 자연스럽게 보여지도록 구성하고 계정 삭제 내의 레이블 그리고 에러 메시지를 수정한 내용 또한 이전보다 자연스럽게 보여지는 것을 확인했다. 사용자 설정 관련 CSS는 얼추 마무리가 된 거 같기 때문에 전체적으로 한 번만 더 확인해 어색한 부분이 있다면 미세조정만 조금 해주면 될 거 같고, CSS에서 주석 처리되거나 중복된 내용은 정리해 깔끔하게 보여지도록 해줄 생각이다. 그리고 뭔가 수정해야 할 부분이 추가로 있었던 것 같은데 현재 기억이 나지 않아서 이 부분은 내용을 수정하면서 생각나면 다시 정리해놓을 예정이며 알림 관련 내용도 간단하게 구성해 기본적인 동작이 될 수 있도록 해줄 생각이다.
