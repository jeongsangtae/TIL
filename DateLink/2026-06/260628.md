      - 친구 목록 조회하는 GET 라우터 수정
        - 친구 목록 확인하는 콘솔 로그 삭제
      - 친구 요청하는 POST 라우터 수정
        - 친구 요청 알림 전송 시에 실행되는 콘솔 로그 삭제
    - group-chat-routes
      - 그룹 채팅방 추가하는 POST 라우터 수정
        - 사용자 정보 확인하는 콘솔 로그 삭제
      - 그룹 채팅방 초대하는 POST 라우터 수정
        - 소켓에 구성된 콘솔 로그 삭제
    - user-routes
      - 회원가입하는 POST 라우터 수정
        - 새 사용자 데이터를 result 변수에 저장한 뒤 MongoDB에 저장하지 않고 바로 저장하도록 수정
        - result 출력하는 콘솔 로그 삭제
      - 로그인하는 POST 라우터 수정
        - 현재 환경 출력하는 콘솔 로그 삭제
  - 프론트엔드

# 오늘 느낀 점

- 백엔드 여러 파일을 확인해보며 아직 삭제하지 못한 주석 처리된 내용과 여러 콘솔 로그 내용들을 삭제해주고 일부 콘솔 로그 내용은 콘솔 에러로 수정해주었는데, 정리가 안된 내용이나 삭제하기에 애매한 내용은 추가로 확인한 뒤 삭제하거나 수정할 예정이다. 그리고 전체적으로 주석이 추가되어 있긴 하지만 큰 틀 안에 어떤 역할을 하는 로직인지 설명하는 주석이 좀 부족해서 이 부분을 추가해줄 예정인데, 우선 백엔드 내용들을 하나씩 확인하면서 주석을 추가할 부분은 추가하고 추가로 삭제하거나 정리가 필요한 부분은 처리해줄 생각이다.
