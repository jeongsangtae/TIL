# 오늘 느낀 점

- 알림 버튼이 우측 상단 투명도 슬라이더 옆에 보여지도록 구성하고 알림 발생 시에 알림 버튼 오른쪽 상단에 빨간점이 보여지며 해당 버튼 클릭 시에 알림 목록이 보여지는 것까지 문제는 없지만, 알림 목록을 열고 확인하는 동안 그대로 유지되는 것이 아닌 타임아웃이 그대로 진행되어 알림 목록이 전부 사라지고 있기 때문에 이에 대한 수정이 필요하다. 생각한 구조는 알림이 발생하면 우측 하단에 보여지는 것과 동시에 알림 버튼 클릭 시에 보여지는 발생한 알림 목록이 그대로 존재하며 알림 버튼 클릭 시에 해당 목록이 드롭다운으로 나열되어 보여지고 다시 알림 버튼을 클릭해 닫을 경우 그 때 사라지는 구조로 구성하는 것을 생각중인데, 현재는 기존 알림 내용을 가져와 발생했을 때 그대로 목록에 보여주는 구조라서 생각한대로 동작하고 있지 않기 때문에 이에 대해 수정을 해줄 생각이다. 기존 알림을 활용해 구성할 수 있으면 최대한 구성해보고 기존 알림을 활용할 수 있다면 알림 관련 정보를 따로 저장해 해당 목록이 보여질 수 있도록 구성할 생각이다.
