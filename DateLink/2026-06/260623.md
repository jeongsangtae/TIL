# 오늘 느낀 점

- 채팅 관련 내용부터 UI 내용까지 살펴보면서 주석 처리된 내용과 불필요한 콘솔 로그를 모두 삭제하고 설명을 위해 추가된 주석 내용은 약간 수정해주었는데 이로써 불필요한 내용이 삭제되어 많이 정리된 것을 확인했고 마지막으로 읽은 메시지 관련 내용은 수정된 주석으로 인해 다음에 다시 구성한다면 깃을 확인해 수정할 생각이다. 그리고 여러 내용을 살펴보면서 콘솔 로그만이 아닌 콘솔 에러와 alert 관련 내용도 있는 것을 확인했는데 여기서 콘솔 에러과 alert 내용을 toast로 수정할 필요는 없는지, 불필요하게 남아있는 것은 아닌지 확인이 필요하며 만약 삭제해도 문제가 없다면 삭제해 정리하고 전체적으로 toast로 에러 관련 처리도 하고 있기 때문에 남겨놓은 내용은 toast로 처리되도록 수정할 생각이다.
