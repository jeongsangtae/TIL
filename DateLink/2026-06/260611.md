# 오늘 느낀 점

- 사용자 설정 모달이 최상단에 보여지도록 여러 테스트를 해봤지만 어떤 부분에서 정확히 문제가 발생했는지 찾지 못했는데, 아마도 여러 CSS가 겹치면서 요소들이 쌓이고 이로 인해 문제가 발생한 것으로 생각이 되며 CSS로 어렵게 해결하기보다 포탈을 사용해 간단하게 해결할 수 있었다. 주석 처리된 내용이나 중복된 내용은 삭제해 정리해주면서 깔끔해졌고 사용자 설정에서 활성화된 버튼이 호버되었을 때와 다른 색으로 보여져 구분이 더 쉬워진 것까지 모두 확인을 해주었는데 이제 알림 관련 내용을 구성해 알림을 놓쳤을 때 알림 버튼을 통해 다시 확인할 수 있도록 내용을 추가해야 하며 그룹 채팅방에서 자신 또는 다른 사용자를 클릭했을 때 보여지는 내용에서 보여지는 아바타와 온라인 점 내용이 재사용될 수 있는지 확인해 해당 내용을 처리해줄 예정이다.
