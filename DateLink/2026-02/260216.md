# 오늘 느낀 점

- 그룹 채팅방 우클릭 시에 열리는 컨텍스트 메뉴 위치가 우클릭한 위치에 열리지 않고 훨씬 아래에 위치한 상태로 열리고 있기 때문에 해당 내용을 수정해 우클릭한 위치에 열리도록 구성하고, 다이렉트 채팅방 목록이나 그룹 채팅방 내의 사용자 목록 그리고 사용자 정보에서 보여지는 아바타 내의 온라인 유무 점 테두리가 좀 더 두껍게 보여져 점이 조금 작게 보이도록 수정을 해봤으나, 생각한 것과 다르게 테두리가 너무 두꺼우면 점이 잘 안보이고 테두리가 너무 얇으면 점이 좀 크게 보여서 중간정도의 두께로 보여지도록 구성하려고 했으나 이것 또한 제대로 해결이 안되어서, 최종적으로 기존에 구성한 상태로 놔두는 것이 제일 나은 판단이라고 생각되어 현재 상태를 유지하기로 결정하였다. 만약 다른 방법을 찾는다면 추가 수정할 수도 있겠지만 별다른 방법이 없다면 현재 상태로 더이상 수정하지 않을 생각이며, 컨텍스트 메뉴 UI가 기본 UI로 보여지고 있는 것 같아서 이 부분은 추가 수정할 생각이다.
