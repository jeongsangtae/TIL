# 오늘 느낀 점

- 그룹 채팅방을 드래그해 다른 채팅방과 겹칠 경우 겹친 채팅방이 바로 이동하지 않고 드래그한 채팅방을 드롭했을 때 이동하도록 구성한 내용은 문제가 없지만, 채팅방을 드래그했을 경우 드래그한 채팅방 기존 위치에 그림자 형태의 아이콘이 보여지도록 구성한 내용은 생각한 것과 다르게 그림자 아이콘이 보이지 않아서 이 내용에 대한 추가 수정이 필요한 상황이다. 현재 생각한 방법으로는 드래그와 관련된 컴포넌트에 구성된 style 변수에서 transform 내용을 주석 처리하거나 none으로 만든 후 어떤식으로 보여지는지 확인하고, 만약 드래그하는 동안 모든 채팅방이 그대로 보여진다면 이 상태에서 드래그 중인 채팅방 기존 위치에 일반적인 그룹 채팅방 아이콘이 아닌 그림자 형태의 아이콘이 보여지도록 CSS를 따로 추가해 테스트해볼 생각이다. 추가로 그룹 채팅방 목록 컴포넌트나 드래그 관련 컴포넌트, 그룹 채팅방 컴포넌트에서 드래그 시에 기존 채팅방 위치에 그림자 형태로 보여질 내용은 어떤 컴포넌트가 적절한 지 다시 확인해볼 생각이며, 드래그 시에 그룹 채팅방 자체를 가지고 이동하는 것이 아닌, 하나의 복사된 채팅방이 드래그되어 원본은 남아있는 형태로 구성하는 것이 가능하다면 해당 방법도 테스트해볼 생각이다.
