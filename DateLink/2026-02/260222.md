# 오늘 느낀 점

- 투명도를 조절하는 슬라이드바 위치가 오른쪽 상단에 위치하는 것보다 로그인한 사용자 정보가 보여지는 부분에 위치하는 것이 더 나은 UI라고 생각되어 위치를 옮기고 CSS도 수정해 확인한 결과, 확실히 이전보다 좀 더 나은 UI와 적절한 위치에서 보여진다고 생각이 들었다. 투명도 조절로 인해 뒤 내용이나 OS 바탕화면을 투영할 수 없기 때문에 해당 내용을 일단 주석 처리하는 것도 고려하긴 했지만 현재 급한 내용은 아니므로 그대로 유지할 생각이며, 프론트엔드에서 백엔드로 전달하는 내용들 일부는 사용하지 않는 내용도 있어서 이 부분은 수정해 사용하지 않는 데이터는 삭제하는 방향으로 진행할 생각이며, 다이렉트 채팅방 목록이 정렬될 때 부드럽게 변경되는 내용에 대해 한 번 더 정보를 찾아본 후 구성해볼 생각이다.
