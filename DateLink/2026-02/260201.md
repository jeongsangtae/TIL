# 오늘 느낀 점

- 그룹 채팅방 목록에서 구성한 드래그 및 드롭 기능에 대해 조금 더 다듬고 필요없는 내용은 모두 정리한 뒤 타입 에러까지 처리해 확인한 결과, 이전보다 훨씬 깔끔해진 상태로 보여지는 것을 확인했는데, 드래그 시에 채팅방이 어떤 위치로 가게 되는지 밑줄 같은게 보여진다면 UX 부분에서 더 향상될 것 같다는 생각이 들어서 추가로 수정할 생각이고, 변경된 그룹 채팅방 목록이 DB에 저장되어 새로고침했을 때나 다시 접속했을 때 또는 채팅방에 들어갔을 때 초기화되지 않고 변경된 상태 그대로 보여지도록 구성해줄 생각이다. 그리고 구성을 하면서 부족한 부분을 발견하면 추가로 수정할 생각이며, UX 향상에 도움이 될 것 같은 내용이 있다면 해당 내용 또한 추가해 구성해줄 생각이다.
