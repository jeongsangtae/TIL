# 오늘 느낀 점

- 테마 모드 프리셋 버튼을 추가하고 내용을 구성해 테스트한 결과, 헤더와 사이드바 그리고 서브 사이드바와 패널 추가로 메인 컨텐츠 내용까지 프리셋에 의해 색이 변경되는 것을 확인했는데, 생각한 것보다 더 만족스러운 UI로 보여지는 것을 확인했기 때문에 추가로 몇 개의 프리셋을 더 추가할 생각이다. 하지만 약간 아쉬운 점으로는 다른 버튼들이나 호버되었을 때 색이 다크 모드에 맞춰서 색이 구성되어 있기 때문에 조금 부족한 부분이 있어서 이 부분은 추가로 수정하는 것을 고려해봐야 한다. 그리고 프론트엔드에서 백엔드로 전달하는 데이터 중에 사용하지 않는 데이터가 있는지 확인해 정리해야 하고, 각 내용들에 주석을 추가해 어떤 용도로 사용되는 내용인지 구분해줄 생각이고, 내용도 다시 확인해 정리가 필요한 내용은 정리를 해줄 생각이다.
