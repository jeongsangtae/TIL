# 오늘 느낀 점

- 활성화된 테마 모드가 로컬 스토리지에 저장되고 새로고침하거나 다시 접속했을 때 저장된 테마 모드 내용을 불러와 적용하도록 구성한 내용이 문제없이 적용되어 더이상 수정하지 않아도 될 거 같고, 이 외에 친구 관련 버튼이나 호버되었을 때 버튼색, 그룹 채팅방 버튼색 내용이 각 테마 모드에 따라서 다른 색이 적용되도록 구성을 해봐야 하고, 친구 관련된 로직도 확인해 정리가 필요한 내용을 정리를 해줄 생각이다. 추가로 사용하지 않는 데이터 추가 확인이 필요하며 이제 필요한 기능들은 다 추가했다고 생각하기 때문에 각 내용별로 어떤 역할을 하는 로직인지 구분이 될 수 있도록 주석을 추가할 생각이다.
