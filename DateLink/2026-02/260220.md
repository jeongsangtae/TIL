# 오늘 느낀 점

- 전체 내용에 적용되는 투명도 관련 내용을 추가해 테스트한 결과, 기대와 다르게 뒤에 보여지는 다른 내용이나 OS 바탕화면이 투명도가 낮을 때 보이지 않고 흰색 배경화면이 보이는 것을 확인했는데, 이건 브라우저에서 제공하는 기본 배경으로 웹 페이지로 구성된 내용에서는 투명도와 배경을 아무리 수정해 테스트해봐도 다른 내용이 보이도록 할 수 없으므로 현재 구성한 내용 위치만 약간 수정해 사용자 정보 공간에 위치하도록 구성할 생각이며, 추가 수정은 eletron 앱으로 변경한 뒤 추가 진행할 생각이다. 그리고 현재 다크 모드로 구성된 내용을 라이트 모드로 변경하는 것은 일단은 보류할 생각이고, 추후에 여러 배경이나 색을 사용자 원하는대로 변경할 수 있도록 내용을 구성할 생각이다.
