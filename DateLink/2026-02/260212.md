# 오늘 느낀 점

- 새롭게 추가한 함수와 액션 내용에서 타입 에러가 발생하면서 이와 관련된 타입 정의를 추가해 해결해주었고, 수정한 타입 정의로 인해 다른 내용에서 또 다른 타입 에러가 발생해 이와 관련된 내용도 수정해 해결을 해주었다. 그리고 실시간 반영과 관련된 액션 내용을 그룹 채팅방과 관련된 Zustand로 옮기는 것을 고려했으나, 옮기는 것을 적절치 못하다고 판단되어 사용자 관련 Zustand에 남겨 그대로 유지할 생각이다. 그룹 채팅방 삭제 또는 추가 시에 사용자 정보에 저장된 그룹 채팅방 목록 내용이 실시간으로 반영되지 않고 있기 때문에 백엔드에 구성된 내용을 확인해 그룹 채팅방이 생성되거나 삭제될 시 해당 그룹 채팅방 ID를 사용자 정보에 저장된 그룹 채팅방 목록 내용에서 추가되거나 삭제되어 업데이트되도록 구성해줄 생각이다.
