# 오늘 느낀 점

- 사용자 프로필 채팅 입력 컴포넌트에서 props 관련 타입 에러가 발생하는 문제를 수정하고, 사용자 프로필 내의 채팅 입력창을 통해 다이렉트 메시지를 보낸 후 입력창이 초기화되어 이전에 전송한 메시지가 남아있지 않도록 수정해주었고, 사용자 프로필을 열었을 때 해당 사용자와 함께 아는 친구 목록이 숫자로 출력될 수 있도록 구성을 해주었다. 사용자 프로필에서 함께 아는 친구 목록 갯수가 출력되는 것은 문제가 없지만 목록 전체를 아직 출력하지 않고 있기 때문에 이와 관련된 내용을 추가해야 하고, 함께 참여한 그룹 채팅방 목록 또한 갯수만 출력하고 전체를 출력하지 않으므로 이 부분 또한 추가를 해야 한다. 같이 아는 친구 목록이나 같이 참여한 채팅방 목록은 사용자 프로필에서 직접 출력하는 것이 아닌, 사용자 프로필 세부 내용을 보여주는 모달에 전달해 해당 내용에서 전체 목록이 보여지도록 구성할 생각이며, 목록 출력 시에 클릭하면 해당 사용자와의 다이렉트 채팅방 또는 해당 사용자 정보로 사용자 프로필 모달이 바뀌도록 할 예정인데 이 부분은 아직 확정된 것이 없기 때문에 추가로 고민한 뒤 구성해볼 생각이다.
