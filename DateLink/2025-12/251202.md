# 오늘 느낀 점

- 그룹 채팅방 이동 시에 이전에 열어놓은 사용자 프로필이 닫히도록 구성한 내용이 문제없이 작동되는 것을 확인했고, 사용자 프로필 외부 공간을 클릭하면 닫히도록 구성한 내용은 작동이 되지만 해당 내용으로 인해 사용자 프로필이 열린 상태에서 해당 사용자를 다시 클릭하면 창이 닫히고 다시 열리는 문제가 발생하는 것을 확인해 수정해봤으나 여전히 오류가 발생하기 때문에 추가 수정을 해줄 생각이다. 해당 내용을 수정하면 사용자 프로필 내에서 보여지는 내용이 자연스러운 위치에서 한 눈에 보이도록 수정할 생각이며, 내용 하나하나에 링크를 추가해 클릭하면 사용자 프로필 모달이 보여지도록 구성해볼 생각이다.
