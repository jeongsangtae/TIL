# 오늘 느낀 점

- 사용자 프로필 채팅 입력창에서 메시지 전달 시에 메시지 받을 사용자 정보를 함께 포함해 관련 액션에 전달하고, 해당 액션에서는 추가로 전달받은 내용에 대한 타입 정의를 추가해 타입 에러가 발생하지 않도록 구성, 액션에서 전달받은 메시지 받을 사용자 정보와 메시지 보낸 사용자 정보 그리고 메시지 내용을 백엔드로 전달해, 백엔드에서 해당 데이터들을 처리해 채팅방 존재 유무에 따라 채팅방을 새롭게 생성해 메시지를 저장하거나, 이미 존재하는 채팅방에 메시지를 저장한 뒤 해당 채팅방이 보여지는 상태로 바꿔 다이렉트 채팅방 목록에서 보여지도록 구성을 해주었다. 사용자 프로필을 통해 전달하는 메시지로 인해 채팅방이 다이렉트 채팅방 목록에서 실시간으로 반영되어 보여지도록 구성이 필요하며, 이미 채팅방이 보일 경우 새롭게 전달된 메시지로 인해 채팅방 순서가 실시간으로 바뀌는 것 또한 추가해야 하고, 마지막으로 다이렉트 채팅방 내에서 사용자 프로필 다이렉트 메시지를 통해 전달되는 메시지가 실시간 반영되어 보여지도록 구성해볼 생각이다.
