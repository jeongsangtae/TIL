# 오늘 느낀 점

- 사용자 프로필에서 사용자 본인은 아바타 이미지 내용만 출력되고, 다른 사용자는 아바타 색만 출력되고 있어서 이와 관련된 구조를 수정해 사용자 본인과 다른 사용자 아바타 이미지 그리고 아바타 색이 모두 출력될 수 있도록 구성해주었으며, 본인 사용자 프로필에서 프로필 편집 버튼 클릭 시에 사용자 프로필 수정창이 열리는 것도 문제가 없었지만 아바타 색모드에서 아바타 색을 따로 선택하지 않고 아바타 이미지 URL을 가진채로 닉네임을 수정 후 제출하면 아바타 이미지 URL이 초기화되는 오류가 발생하기 때문에 관련 액션에 구성된 조건부를 세세하게 구성해 아바타 색모드에서 아바타 색이 없고 아바타 이미지 URL만 있는 상태에서 닉네임을 업데이트하면 기존 아바타 이미지 URL이 그대로 적용되어 보여지며 닉네임은 업데이트되도록 수정해주었다. 원래라면 다이렉트 메시지를 보낼 수 있도록 구성하려고 했으나 기존 내용을 수정하고 약간의 내용을 추가하던중 오류를 발견하게 되어 해당 오류를 먼저 수정해주었으며, 다시 다이렉트 메시지 관련 내용을 구성해 테스트해볼 생각이다.
