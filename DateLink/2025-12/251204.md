# 오늘 느낀 점

- 사용자 프로필 창이 열린 상태에서 해당 사용자를 다시 클릭할 경우 창이 닫힌 다음 다시 열리는 문제가 발생했고 이로 인해 관련 조치를 했지만 제대로 해결이 되지 않아 추가로 수정한 결과, 이제는 문제가 해결되어 사용자 프로필 외부를 클릭하면 창이 닫히며 사용자를 다시 클릭해도 사용자 프로필 창이 다시 열리지 않고 그대로 닫히는 것을 확인할 수 있었다. 이제 사용자 프로필에서 보여줘야 하는 함께 아는 친구 목록 갯수와 여러 내용 배치 그리고 내용들을 클릭할 수 있도록 구성해 클릭 시에 사용자 프로필 모달이 보여지도록 구성할 생각이다.
