# 오늘 느낀 점

- 사용자 프로필에서 열리는 사용자 프로필 수정 폼 또한 그룹 채팅방 세부 내용에 옮겨 열리도록 구성하고 사용자 프로필 수정 폼이 열릴 때 사용자 프로필이 닫히지 않는 문제를 해결하기 위해 사용자 프로필 수정 폼 관련 내용 또한 그룹 채팅방 세부 내용에서 상태 끌어올리기로 데이터를 전달받아 사용하도록 구성한 결과, 사용자 프로필 수정 폼이 열릴 때 사용자 프로필이 닫혀 보이지 않는 것을 확인했다. 그리고 그룹 채팅방 세부 내용에서 사용자 프로필 수정 폼이나 세부 내용을 열도록 구성하면서 새롭게 추가한 내용들 타입 에러가 발생해 재사용 타입 정의를 수정하고 해당 내용을 사용해 타입 에러를 모두 해결했는데, 아직 사용자 프로필 세부 내용에서 발생한 타입 에러는 수정하지 않은 상태라서 이 부분 또한 수정해줄 생각이다. 그리고 다이렉트 채팅방 패널 또는 사용자 프로필 세부 내용에서 친구 클릭 시에 해당 채팅방으로 이동하는 내용에서 채팅방이 없을 경우 어떤식으로 작동하는지 확인을 하지 않은 상태기에 확인하고 어떤식으로 조치할지 결정할 생각이다.
