# 오늘 느낀 점

- 단순하게 사용자 프로필 세부 내용이 열리면 사용자 프로필이 닫히는 구조로 구성을 해보려고 했으나, 생각보다 구조가 그렇게 간단히 구성되어 있지 않아서 사용자 프로필 세부 내용이 열리면서 사용자 프로필이 함께 닫히는 문제가 발생해, 사용자 프로필 세부 내용 열리는 내용을 그룹 채팅방 세부 내용에 옮기고 관련 정보를 사용자 프로필에서 상태 끌어올리기로 전달받아 테스트한 결과, 사용자 프로필 세부 내용이 열릴 때 사용자 프로필이 닫히도록 구성할 수 있었다. 하지만 사용자 프로필 수정 폼은 열릴 때 사용자 프로필이 닫히지 않는 문제가 발생하고 있기 때문에 이에 대한 추가 수정을 해줄 생각이다. 그리고 아직 사용자 프로필 세부 내용에서 함께 아는 친구 클릭 시에 채팅방이 존재하지 않으면 어떤식으로 처리되는지 확인을 해보지 못했기 때문에 확인 후 처리해야 하며, 다이렉트 채팅방 패널에서도 어떤식으로 처리되는지 확인해 조치해줄 생각이다.
