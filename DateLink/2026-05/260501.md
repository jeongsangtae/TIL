# 오늘 느낀 점

- 사용자 비밀번호 수정 폼 액션에서 사용자가 입력한 현재 비밀번호, 새로운 비밀번호, 새로운 비밀번호 확인, 메소드 정보를 전달받도록 구성하고 해당 내용을 객체 디스트럭처링한 뒤 객체로 묶어 백엔드로 전달되도록 구성했지만 정작 입력한 비밀번호 관련 내용은 전달하지 않은 상태라서 이와 관련된 내용을 수정할 예정이며, 사용자 설정에 구성된 비밀번호 입력 내용을 사용자 비밀번호 수정 폼에 옮겨주었고 비밀번호 변경 버튼을 클릭하면 사용자 설정 내용에서 사용자 비밀번호 수정 폼 내용으로 전환되어 보여지고 반대로 뒤로가기 버튼 클릭 시에 사용자 설정 화면으로 다시 전환되도록 구성했는데 현재까지 구성한 내용은 모두 문제없이 적용되고 있기 때문에 UI 관련만 수정해주면 될 것 같고, 그리고 비밀번호 입력과 관련된 내용이 액션 함수에 전달되고 그 정보가 백엔드에 전달되어 비밀번호가 업데이트되도록 구성할 생각이다.
