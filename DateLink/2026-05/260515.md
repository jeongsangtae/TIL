# 오늘 느낀 점

- 서브 사이드바에서 많은 내용이 보여질 때 스크롤이 생겨나는데 이 때 스크롤이 상단에 있을 경우 그룹 초대 같은 내용이 사용자 정보 뒤로 가려진 것처럼 보여지는데 이 문제를 해결하기 위해 서브 사이드바 스크롤 전용 div를 추가하고 해당 내용에 스크롤바 관련 내용을 구성한 결과 깔끔하게 문제가 해결되었고, 로그아웃과 계정 삭제 시에 로컬 스토리지에 저장된 마지막으로 읽은 메시지 항목들이 모두 삭제되도록 구성한 내용 모두 문제없이 작동된 것도 확인을 해주었는데, 로그인 시에 마지막으로 읽은 메시지를 불러오는 내용은 아직 구성이 부족한 상태라서 추가 구성 후 테스트해볼 예정이다. 그리고 현재 마지막으로 읽은 메시지 내용 ID를 DB에서 저장하고 있는데 로컬 스토리지에서는 해당 ID 대신 위치 번호로 기억하고 있어서 이걸 수정해 DB에서 저장한 내용 그대로 불러와 로컬 스토리지에 저장하고 마지막으로 읽은 메시지 ID 기준으로 다시 채팅방으로 돌아왔을 때 정확한 위치로 스크롤이 이동하도록 수정해볼 생각이다.
