# 오늘 느낀 점

- 사용자 설정에서 계정 삭제 폼에 토글 프롭스를 전달해 실행되도록 구성하고 타입 정의도 수정해 타입 에러가 발생하지 않도록 구성해주었으며 서브 사이드바에서 내용이 많아질 경우 스크롤바가 보여지는데 이 스크롤바가 기본 UI로 구성되어 있어서 좀 더 자연스럽게 보여지도록 UI를 수정하고 스크롤바가 최상단에 있을 경우 사용자 정보 뒤로 그룹 초대 내용이 가려지는 것처럼 보여지는 문제가 발생해 수정해주었지만 여전히 수정해도 스크롤바가 최상단에 있으면 사용자 정보 뒤로 그룹 초대 내용이 가려지는 문제가 발생해 이와 관련해 추가 수정할 예정이다. 스크롤이 최하단에 있을 경우에는 사용자 정보 위에 그룹 초대 내용이 보여지면서 자연스럽지만 스크롤이 최상단에 있을 경우엔 내용이 하단까지 내려가버리기 때문에 이에 대한 수정이 필요한데, 이걸 어떻게 수정하는 것이 좋을지 아직 감이 잡히지 않아서 여러 방법으로 테스트해볼 생각이며 계정 삭제나 로그아웃 시에 로컬 스토리지에 저장된 마지막으로 읽은 메시지 내용이 삭제되지 않는 문제가 발생하므로 이에 대한 처리도 해줄 생각이다.
