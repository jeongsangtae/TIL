# 오늘 느낀 점

- 사이드바에서 보여지는 사용자 정보 내용이 잘려서 보이는 문제와 그룹 채팅방이 사용자 정보에 가려져 클릭이 안되는 문제 그리고 사이드바에서 스크롤 이동 시에 사용자 정보가 함께 이동하는 문제는 CSS 수정으로 모두 해결이 되었지만 그룹 채팅방 드래그를 좌우로 이동 시에 빈 공간이 생기는 문제는 해결할 수 있긴 하지만 이 문제를 해결하면 그룹 채팅방에 구성된 인디케이터와 툴팁 메시지 관련 내용이 보이지 않는 문제가 발생하므로 추가 수정이 필요한 상황이다. 현재 생각한 방법으로는 인디케이터와 툴팁 메시지를 다른 쪽에서 출력하도록 구성하는 것과 사이드바 스크롤을 없애고 그 상태에서 좌우 드래그 시에 어떤식으로 작동하는지 확인해 이와 관련된 내용을 수정하는 것을 고려중인데, 여기에 드래그 관련 내용도 추가로 확인해 이 부분에서 문제가 있는지 확인해 수정한 뒤 테스트해볼 생각이다. 일단 발견한 문제들부터 수정한 뒤 채팅 목록에 구성된 마지막으로 읽은 메시지 관련 스크롤 내용을 마지막으로 수정해 테스트해보고 사용자 설정과 관련된 UI를 수정할 예정이다.
