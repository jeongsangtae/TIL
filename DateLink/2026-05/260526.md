# 오늘 느낀 점

- 마지막으로 읽은 메시지 관련 내용은 모두 주석 처리하고 간단한 설명 주석을 추가해 다음에 다시 사용될 수 있도록 해주었으며 채팅방 입장 시에 스크롤이 최하단으로 이동하는 내용과 새로운 메시지 또는 사용자가 메시지를 추가했을 때 스크롤이 제어되어 되는 내용을 추가했는데 모두 문제없이 작동하는 것을 확인했다. 스크롤 관련 내용을 여러 방법으로 수정하고 테스트해봤지만 의도대로 작동하지 않아서 조금 단순하게 수정했는데 다음에 좋은 방법을 찾거나 좀 더 여유가 생긴다면 다시 수정해볼 예정이다. 사용자 설정 관련 UI를 처리해야 하는데 이 내용에서 왼쪽에 선택하는 부분은 다른 서브 사이드바 내용처럼 보여지도록 수정할 생각이고 오른쪽 메인 내용은 좀 더 자연스럽게 보여지도록 수정할 생각인데 이 부분은 수정을 해보면서 최대한 자연스럽게 보이도록 구성할 예정이다.
