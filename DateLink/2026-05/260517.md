# 오늘 느낀 점

- 마지막으로 읽은 메시지 위치를 감지하는 내용이 이전보다 오차가 줄어든 것을 확인했는데 반환하는 내용에서 이전에는 안 보이는 메시지까지 감지해 위치를 읽었지만 지금은 컨테이너에서 보이는 위치만 읽어서 감지해 관련 정보를 전달하기 때문에 이전보다 확실히 오차가 줄어든 상태로 감지하는 것을 확인했다. 그리고 로그인 시에 마지막으로 읽은 메시지 정보를 불러오기 전에 메시지 갯수를 저장하는 내용부터 추가했는데 해당 정보가 없다면 마지막으로 읽은 메시지 정보를 불러와도 값이 없어서 생각한대로 동작이 되지 않기 때문에 해당 내용을 먼저 구성해주었다. 필요한 내용은 얼추 구성이 된 것 같아서 로그인 시에 마지막으로 읽은 메시지 정보를 불러오는 내용을 구성해줄 예정이며 이번에 수정하면서 추가한 콘솔 로그 정리 및 타입 에러 관련 내용은 수정해야 하고 그룹 채팅방 목록에서 그룹 채팅방 갯수가 많아질 때 아래 잘리는 것 같기 때문에 이와 관련된 문제도 수정해줄 예정이다.
