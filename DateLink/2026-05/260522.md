# 오늘 느낀 점

- 사이드바에서 처리되지 않은 여러 문제를 해결하기 위해 추가 수정한 뒤 다시 확인해본 결과, 사용자 정보 내용만의 문제가 아닌 그룹 채팅방에 커서를 올리면 보여지는 인디케이터와 툴팁 메시지 문제도 발견되어 이 부분에 대한 추가 수정이 필요하다. 현재 생각한 방법으로 그룹 채팅방에서 사용자 프로필이나 다른 사용자 프로필 여는 기능처럼 특정 내용에 커서를 올리거나 활성화했을 때 그 내용 위치에 따라서 인디케이터와 툴팁 메시지가 보여지도록 구성을 해볼 생각인데, 이 내용이 잘 될 지는 확신이 없긴하지만 그래도 시도해볼 생각이며 이 방법 전에 비슷한 수정 방법으로 포지션 내용을 fixed로 변경해 위치가 조정되도록 테스트해볼 생각이다.
