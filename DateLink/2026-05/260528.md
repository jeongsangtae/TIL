# 오늘 느낀 점

- 채팅 메시지 옆에 ID가 출력되지 않도록 수정하고 채팅 메시지 ID와 관련된 내용들을 삭제해 타입 에러도 해결하고 사용자 설정과 사용자 프로필 내용 관련 CSS를 수정해 이전보다 자연스럽게 보여지도록 구성하긴 했지만 아직 뭔가 부족한 느낌이 들어서 추가 수정할 예정이며 계정 설정 내용인 비밀번호 수정과 계정 삭제 관련 내용은 아직 CSS를 수정하지 못했기 때문에 이 부분 또한 수정해 자연스러운 UI로 보여지도록 할 생각이다. 그리고 사용자 프로필 내용이 이전에 구성한 내용 그대로 가져와 모달이 없어진 형태라서 뭔가 공간이 남고 조금 부자연스러운 UI로 보여지고 있기 때문에 추가 수정이 필요한데, 이 내용을 어떤식으로 수정하는 것이 좋을지 감이 잡히지 않아서 좀 더 생각이 필요하며 전체적으로 내용을 수정하는 것보다는 빈 공간을 채우거나 기존 내용을 약간 키우는 방식으로 진행할 생각이다.
