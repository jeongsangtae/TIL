# 오늘 느낀 점

- 계정 삭제 화면에서 비밀번호 입력창에 구성된 눈 아이콘을 클릭하면 암호화된 비밀번호가 텍스트로 변경되어 보이지 않고 그대로 암호화되어 보여지고 있었기 때문에 해당 내용을 수정해 암호화와 텍스트 전환이 되도록 구성하고 계정 삭제 라우터에서 다이렉트 채팅 메시지와 호스트인 그룹 채팅방 메시지 내용이 따로 구성되지 않고 하나의 내용에서 함께 삭제되도록 수정 및 그룹 채팅방 초대와 마지막으로 읽은 메시지 내용이 계정 삭제 시에 함께 삭제되도록 구성하고 테스트해본 결과, 계정 삭제 시에 삭제하려고 하는 내용들은 모두 삭제되고 계정도 삭제되긴 했지만 중간에 DB에서 오류가 발생해서 이에 대한 추가 확인이 필요한 상황이다. 무엇 때문에 오류가 발생하는지 확인해 수정할 예정이며 테스트 중 로컬 스토리지에 저장된 마지막으로 읽은 메시지 내용이나 그룹 채팅방 초대 내용이 약간 잘린 형태로 보이지 않는 부분까지 수정해 처리해줄 예정이다.
