# 오늘 느낀 점

- 사용자 프로필 세부 내용 내의 친구 확인, 메시지 버튼에 커서를 올리면 툴팁이 보여지도록 내용을 수정했는데 구성한 내용이 문제없이 적용되어 아이콘이 아닌, 버튼에 커서를 올리면 툴팁이 보여지는 것을 확인했고 툴팁 내용 또한 문제없이 출력되어 더이상 추가 수정은 하지 않아도 될 것 같다. 사용자 프로필 세부 내용에서 친구가 아닌 사용자의 경우 텍스트로 보여주고 있는데 이 내용에 아이콘이나 보조 설명을 추가해 UX를 향상 시켜볼 생각이며 여러 아바타 내용의 온라인 유무 테두리 두께 또한 수정해 좀 더 온라인 관련 점 내용이 확실히 눈에 띄도록 해줄 생각이다. 이 외에 전체 화면 투명도, 그룹 채팅방 위치 변경, 그룹 채팅방 컨텍스트 메뉴 관련 내용 등 여러 내용 수정이 필요하므로 빠르게 진행해볼 생각이다.
