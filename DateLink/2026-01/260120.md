# 오늘 느낀 점

- 사용자 프로필 세부 내용에 사용할 여러 아이콘을 추가하고 해당 아이콘을 사용해 메시지 버튼과 친구 확인, 친구 추가 버튼이 좀 더 자연스럽게 보이도록 수정하고 친구된 기간이나 친구가 아닌 사용자 내용이 해당 영역을 벗어나지 않고 이전보다 자연스럽게 보이도록 구성을 했는데, 전체적으로 만족스럽긴 하지만 친구가 아닌 사용자를 텍스트로 보여주고 있는데 조금 허전한 느낌이 들어서 추가로 수정이 필요하고, 친구 확인과 친구 추가 아이콘 그리고 버튼이 CSS를 추가해도 뭔가 부자연스러운 UI로 보여지고 있기 때문에 추가로 수정할 예정이며, 메시지와 친구 추가 또는 친구 확인 버튼에 커서를 올리면 버튼만 호버 스타일로 바뀌는 것이 아닌, 상단에 내용이 추가로 보여지도록 구성을 해볼 생각이다. 사용자 프로필 세부 내용 UI도 얼추 수정이 끝나가는데, 모두 마무리한 뒤 그룹 채팅방 위치 변경이나 우클릭 시 보여지는 컨텍스트 메뉴 관련 UI 등 여러 내용을 추가로 수정할 생각이다.
