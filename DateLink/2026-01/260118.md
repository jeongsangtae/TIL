# 오늘 느낀 점

- 사용자 프로필 세부 내용에서 친구가 아닌 사용자의 경우 친구 추가 아이콘 버튼이 보여지는데 이 때 해당 버튼을 클릭하면 사용자에게 실시간으로 친구 요청이 넘어가도록 구성한 내용이 문제없이 실행되어 친구 요청 목록에 실시간 반영되는 것을 확인했고 같이 아는 친구 아바타와 닉네임이 한 줄로 보여지며 친구 아바타 또는 닉네임에 커서를 올리면 영역 전체에 호버 효과가 보여져 변화하는 것까지 모두 적용되어 이전보다 훨씬 나은 UI로 보여지는 것을 확인했다. 같이 아는 친구 내용은 이제 생각한대로 모두 보여지는 것 같아서, 같이 참여한 채팅방 관련 아이콘도 보여지도록 구성해 자연스럽게 보여지도록 구성할 생각이며, 친구가 아닌 사용자 프로필 세부 내용에서 친구 추가 아이콘 버튼이 텍스트 대신 아이콘 버튼으로 보여지도록 수정이 필요하며 메시지 버튼과 한 줄로 자연스럽게 보이도록 수정할 생각이다. 그리고 친구인 사용자와 친구된 기간 내용과 친구가 아닌 사용자일 때 보여지는 텍스트 내용이 자연스러운 UI와 위치에서 보여지도록 수정할 생각이다.
