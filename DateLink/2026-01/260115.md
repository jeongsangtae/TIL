# 오늘 느낀 점

- 사용자 프로필 세부 내용에서 같이 아는 친구와 같이 있는 그룹 채팅방 버튼이 정확히 텍스트를 클릭했을 때만 활성화될 수 있도록 수정하고, 호버 또는 활성화되었을 때 밑줄이 영역 전체로 보여지는 것이 아닌 버튼 텍스트 길이만큼만 보여지도록 구성해 이전보다 좀 더 자연스러운 UI로 보여지도록 수정을 해주었다. 사용자 프로필 세부 내용에서 해당 사용자와 친구된 기간을 보여주도록 구성할 생각인데, 만약 친구가 아닐 경우 친구가 아님을 알려주며 바로 친구 추가할 수 있도록 버튼을 추가할 예정이다. 그리고 사용자 프로필 세부 내용에서 같이 아는 친구와 같이 참여한 채팅방 목록이 보여지는 공간에서 친구 아바타 또는 채팅방 아이콘이 보이고 있지 않으므로 해당 내용을 추가해 보여지도록 구성해볼 생각이다.
