# 오늘 느낀 점

- 사용자 프로필 세부 내용 구조를 수정하고 CSS를 추가해 확인한 결과, 이전보다는 나아졌지만 여전히 UI가 부족한 느낌이 많이 들어서 추가로 수정을 해야 하는데, 왼쪽 1/3 내용은 아바타 헤더와 아바타 그리고 닉네임과 메시지 버튼이 보이고 있는데 아래 부분에 공간이 많이 남아서 이 공간을 어떻게 활용할 지 생각을 해봐야 하며, 오른쪽에는 같이 아는 친구 목록과 같이 참여한 채팅방 목록이 보여지는데 이 내용은 위치만 잡고 별도의 CSS를 추가하지 않은 상태라서 CSS 내용을 구성해 어떤식으로 보이는지 확인이 필요하고, 같이 아는 친구와 같이 참여한 채팅방 버튼 활성화 시에 활성화되었다는 표시를 조금 다르게 구성해 좀 더 눈에 띄도록 해볼 생각이다.
