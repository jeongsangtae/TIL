# 오늘 느낀 점

- 사용자 프로필 세부 내용 구조와 CSS를 수정해 확인한 결과, 아바타와 사용자 내용이 보여지는 영역이 모달창 배경과 약간 공간이 생겨 좀 더 자연스러운 UI로 보여지는 것을 확인했고, 함께 아는 친구나 함께 참여한 채팅방 버튼 활성화 시에 이전과 다르게 활성화된 버튼이 확연히 눈에 띄게 보여져 구분이 쉬워진 것을 확인했다. 그리고 사용자 프로필에서 보여지는 온라인 유무 점 테두리 CSS가 변경되면서 이전보다 좀 더 나은 UI로 보여지는 것을 확인했는데, 다른 아바타에서 보여지는 온라인 유무 점 내용은 테두리 두께가 약간 작아서 뭔가 UI상 조금 부족하게 느껴지고 있으므로 해당 내용도 수정해 좀 더 자연스럽게 보이도록 수정할 생각이다. 추가로 사용자 프로필 세부 내용에서 함께와 관련된 내용 활성화 시 아래줄에 선이 보여지는데 이 선이 현재 활성화되었을 때만 보여지고 있는 상태이므로 해당 내용을 수정해 전체적으로 선이 보여지고 활성화된 내용 아래 선은 더 굵은 느낌의 선으로 보여지도록 구성해볼 생각이다.
