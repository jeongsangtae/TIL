# 오늘 느낀 점

- 사용자 프로필 세부 내용에서 테스트로 사용될 내용 관련 프롭스를 삭제해 정리해주었고, 같이 참여한 채팅방 아이콘이 보여지며 해당 내용이 자연스럽게 보이도록 CSS 구성 및 같이 아는 친구 아이콘 크기와 내용이 같이 참여한 채팅방 아이콘 크기처럼 보여져 내용이 전환될 때 이질감이 들지 않도록 수정한 내용 모두 문제없이 적용되어 이전보다 훨씬 자연스러워진 것을 확인할 수 있었다. 친구된 기간 또는 친구가 아니라는 텍스트 메시지가 자연스럽게 보이도록 수정이 필요하고, 메시지 버튼과 친구 버튼 또는 친구 추가하기 버튼이 일반적인 텍스트가 아닌 아이콘 형태로 보여져 더 자연스럽게 보이도록 수정이 필요하며 다이렉트 채팅방 목록, 그룹 채팅방 내의 사용자 목록, 등에서 보여지는 아바타 내의 온라인 점 테두리 두께 설정을 수정해 다른 내용에서 보여지는 온라인 점과 유사하게 보이도록 구성해줄 생각이다. 이 외에 컨텍스트 메뉴 관련 내용이나 화면 투명도 내용 등 여러 내용이 더 있지만 일단 앞에서 처리해야 하는 것부터 처리하고 이후에 추가로 수정할 예정이다.
