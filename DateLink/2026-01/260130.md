# 오늘 느낀 점

- 그룹 채팅방 목록에서 그룹 채팅방 드래그 후 드롭하면 해당 채팅방이 클릭되는 문제가 발생했었는데, 해당 문제 관련 여러 테스트를 진행해봤지만 해결하지 못해 추가로 내용을 수정해 테스트를 진행한 결과, 이제는 문제가 해결되어 드래그 후 드롭 시에 채팅방이 클릭되지 않고 그대로 유지되는 것을 확인했다. 여기서 UX를 좀 더 향상시키기 위해 드래그를 할 때는 커서가 포인터가 아닌 드롭 형태로 보이도록 구성해야 하고, 드래그해서 드롭할 때 어떤 위치에 옮기는지 그 아래 구분선이 보이면 더욱 UX가 향상되어 도움이 될 것 같기에 해당 내용 또한 추가할 생각이다. 그리고 현재 테스트를 위해 추가한 여러 내용들이 주석 처리되어 있고, 사용중인 내용도 있는데 여기서 확실하게 필요한 내용만 남기고 불필요한 내용은 모두 삭제해 정리할 생각이며, 드래그와 드롭으로 변경된 그룹 채팅방 목록이 DB에 저장되어 새로고침하거나 다시 접속했을 때 이전에 변경한 순서대로 보이도록 관련 내용을 추가할 생각이다.
