# 오늘 느낀 점

- 사용자 프로필 세부 내용 내의 친구 확인, 친구 추가 버튼 관련 구조와 CSS를 수정해 UI가 개선되도록 구성한 내용은 문제가 없지만, 친구 확인과 메시지 버튼 툴팁 관련 내용은 추가해 테스트해봤지만 생각한 것과 다르게 제대로 출력이 되지 않아서 추가 수정이 필요한데, 이 툴팁 내용들은 이전에 그룹 채팅방 툴팁을 구성했던 내용을 참고해 다시 수정해볼 생각이다. 그리고 친구가 아닌 사용자의 경우 친구된 기간이 아닌 단순히 텍스트로만 구성된 내용이 보여지고 있는데 이 부분이 많이 허전하다고 생각되어 추가로 좀 더 확실하게 보이거나 이전보다 나은 UI로 보여질 수 있도록 수정을 해봐야 하며, 여러 내용의 아바타 관련 내용도 수정을 해야 하는데 UI 관련 구조와 CSS를 구성하는 부분에서 내 생각과 다르게 원하는 결과물이 빠르게 나오지 않아 조금 더뎌지는 느낌이 있어서 적당히 타협해 좀 더 빠르게 구성 및 수정해 진행할 생각이다.
