# 오늘 느낀 점

- 그룹 채팅방 목록에서 그룹 채팅방 드래그 후 드롭하면 해당 채팅방이 클릭되어 이동되는데, 해당 문제를 해결하기 위해 내용을 수정해봤으나 생각한 것과 다르게 원인이 제대로 해결되지 않아서 이전과 같은 동일한 문제가 발생하고 있기 때문에 추가 수정이 필요한 상황이다. 어떤식으로 처리를 해야 드래그와 드롭 시에 채팅방이 클릭되지 않고 채팅방 목록만 변경할 수 있을지 감이 안잡혀서 좀 더 정보를 찾아봐야 하는데, 한 가지 생각해본 내용으로는 앞에서 구성한 useRef 내용이 콘솔에서 계속 true로 반영되고 있는데 이 내용이 false로 넘어오는 것이 문제없어진다면 처리할 수 있을 것 같다는 생각이 들어서 내용 수정 후 다시 테스트해볼 생각이며, 드롭했을 때 자동으로 클릭이 되지 않도록 내용을 추가 구성하는 것도 방법이 될 수 있을 것 같아서 해당 방법 또한 추가 수정해 테스트해볼 생각이다. 확신은 없지만 그래도 몇 가지 테스트할 방법이 생각나서 해당 방법들을 시도해볼 생각이며, 안되면 정보를 더 찾은 뒤 테스트해볼 생각이다.
