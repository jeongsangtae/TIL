# 오늘 느낀 점

- 사용자 프로필 입력창에서 높이를 늘리고 스크롤이 생기게 한 다음, 다이렉트 메시지를 전송하면 해당 내용이 초기화되어 초기 상태로 돌아가도록 구성하고, 스크롤 UI 관련 CSS도 수정해 적은 공간에서 자연스럽게 보이도록 구성 및 줄바꿈으로 스크롤이 생겨날 때 스크롤이 최하단에 위치해 내용이 잘리지 않도록 구성한 내용 모두 크게 문제는 없지만, 초기에 줄바꿈을 적용하면 내용이 살짝 위로 올라가는 현상을 발견해서 이와 관련해 수정해줄 생각이다. 그리고 사용자 프로필에서 입력창을 늘리거나 내용을 입력하고 다른 사용자 프로필 클릭 시에 그대로 내용이 유지된 채로 보여지고 있기 때문에 이 내용 또한 수정을 해줘야 하며, 사용자 프로필 입력창에서 메시지 전달 후 초기화만 되고 별다른 동작이 없는데, 여기에 메시지 전송 후 해당 다이렉트 채팅방으로 이동하는 구조를 추가할 지 고민중인데 이 부분은 좀 더 고민을 해보고 추가할 지 아니면, 사용자 프로필 세부 내용에서 해당 다이렉트 채팅방으로 이동하는 버튼으로 사용자가 이동을 원할 때만 이동하도록 할 지 확실히 정한 후 구성할 생각이다.
