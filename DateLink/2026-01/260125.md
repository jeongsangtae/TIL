# 오늘 느낀 점

- 사용자 프로필 세부 내용 내의 친구가 아닌 사용자 관련 텍스트 내용이 좀 더 UX가 향상된 채로 보여지도록 구성한 내용이 만족스러워서 더이상 추가 수정은 필요하지 않을 것 같고, 여러 아바타에서 보여지는 온라인 유무 점 관련 테두리를 수정해 좀 더 UI를 향상시켜봐야 하고, 그룹 채팅방 위치 변경과 우클릭 시에 보여지는 컨텍스트 메뉴 위치와 UI를 수정하고, 전체 화면 투명도 관련 내용도 추가해 테스트해줄 예정이다. 그리고 사용자 프로필에서 메시지 전송 후 다이렉트 채팅방으로 이동하는 로직은 일단은 추가하지 않는 쪽으로 생각하긴 했는데, 또 다르게 생각하면 메시지 전송 후 이동되지 않으면 메시지가 전송되었다는 것을 확인할 방법이 없어서 이동하는 쪽이 좋아보이기도 하지만, 메시지 전송 후 메시지가 전송되었다는 안내 문구같은게 뜨면 문제가 없을 것 같다는 생각도 들어서 초기에 구성한대로 이동하지 않고 문구가 보여지는 쪽으로 구성을 해볼 생각이다.
