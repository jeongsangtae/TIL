# 오늘 느낀 점

- 사용자 프로필, 사용자 프로필 메시지 입력창, 다이렉트 채팅방 패널, 그룹 채팅방 패널 등 여러 내용에 구성된 CSS를 수정하고 새롭게 추가해 확인한 결과, 이전보다 확실히 나아진 형태로 보이기는 하지만 조금 부족한 느낌이 들어서 추가로 수정해줄 생각이다. 사용자 프로필 세부 내용 UI 또한 현재 CSS가 없는 상태로 기본 UI로 보여지기 때문에 이 내용 또한 CSS를 추가해 처리해줄 생각이고, 그룹 채팅방 우클릭 시에 보여지는 컨텍스트 메뉴 UI가 다른 내용과 어우러지지 못하고 부자연스럽다는 생각이 들어서 추가로 수정할 생각이며, 컨텍스트 메뉴 열리는 위치도 조금 이상한 것 같아서 확인이 필요하고 사이드바에서 보여지는 그룹 채팅방 위치를 위아래로 자유롭게 이동시켜 순서를 바꿀 수 있도록 구성해볼 생각이다.
