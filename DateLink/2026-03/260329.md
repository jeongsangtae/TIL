# 오늘 느낀 점

- 패딩 관련 CSS 내용을 다시 확인한 뒤 변수화하기 애매했던 값들은 CSS 공용 변수로 사용되는 수치로 수정해 CSS 공용 변수를 적용시키고 UI를 해칠만한 내용은 굳이 변수화하지 않고 그대로 남겨주었다. 원래는 마진 관련 내용까지 모두 함께 수정해주려고 했으나 패딩 관련 내용을 하나씩 테스트해보면서 확인하느라 시간이 조금 길어져서 마진은 이제 다시 확인해 수정해줄 예정이다. 그리고 #555 같은 색 내용이 여러곳에서 중복으로 사용되고 있는 것을 확인했기 때문에 해당 내용도 변수화 작업을 해줘야 하는데, 이런 내용처럼 더 변수화할 내용이 없는지 확인해보고 추가로 변수화할 내용이 있다면 작업해 적용해줄 생각이다.
