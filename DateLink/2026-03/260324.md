# 오늘 느낀 점

- 테두리 관련 빠진 내용을 CSS 공용 변수에 추가하고 해당 내용을 포함한 여러 테두리 공용 변수를 적용해 반영하고 스크롤바 내용도 마찬가지로 반영해 확인한 결과, 모두 문제없이 적용되어 보이는 것을 확인했으며 다이렉트 채팅방 패널에 구성된 테두리 내용이 제대로 적용되지 않는 문제를 발견해 해당 내용은 수정해주면서 해결이 되었지만, 다이렉트 채팅방 패널과 그룹 채팅방 패널에서 내용이 길어질 경우 스크롤이 생겨나도 아래가 잘리는 문제가 발생해 이와 관련된 문제를 추가로 수정해줘야 한다. 그리고 다이렉트 채팅방 패널이 길어질 경우 스크롤이 보여지게 되었는데 이 때 스크롤이 기본 UI로 보여지고 있기 때문에 해당 내용은 좀 더 자연스러운 UI로 보여지도록 수정해줄 생각이다. CSS 내용을 추가로 확인해 변수화해야 할 내용이 있는지 더 찾아봐야 하며, 만약 없다면 이정도에서 마무리하고 마지막 확인을 한 뒤 전체적인 주석을 추가하고 배포하는 방향으로 진행할 생각이다.
