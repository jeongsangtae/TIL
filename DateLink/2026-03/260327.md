# 오늘 느낀 점

- 공간 관련 내용인 패딩과 마진 그리고 갭 내용에 대해 미리 구성한 CSS 공용 변수로 전부 적용시켜주었는데 해당 내용들은 모두 문제가 없었지만 몇몇 내용들이 CSS 공용 변수로 수정할 수 없는 값으로 구성되어 있어서 이 내용들을 확인해 공용 변수로 변경해도 문제가 없는지 추가 테스트가 필요한 상황이다. 애매한 값으로 구성된 내용들이 조금 있어서 최대한 공용 변수의 값으로 적용될 수 있도록 수정할 생각이며 우선 값을 변경해 화면에서 어떻게 보이는지 테스트해본 뒤, 큰 문제가 없다면 공용 변수 값으로 변경하는 작업을 해줄 생각이다. 그리고 변수화하지 않는 내용이 있는지 확인이 필요하기 때문에 전체적으로 한 번 더 점검을 해준 뒤 처리해줄 생각이다.
