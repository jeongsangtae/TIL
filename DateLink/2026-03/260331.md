# 오늘 느낀 점

- 사용자 프로필 세부 내용에서 보여지는 스크롤과 같이 아는 친구 그리고 같이 참여한 채팅방 관련 CSS를 수정해 좀 더 자연스럽게 보이지도록 해주었으며, 전체 내용에서 스크롤을 클릭한 상태로 활성화시키면 검은 형태로 보여져 다른 내용과 구분이 되지 않아 잘 보이지 않는 것을 확인했는데 활성화 색을 호버되었을 때 색과 동일하게 구성해주면서 이제는 확실하게 구분되어 보여지는 것을 확인해주었다. 사용자 정보에서 보여지는 슬라이드바 내용으로 인해 닉네임이 길어질 경우 닉네임이 줄여진 형태로 보이지도 않고 줄여져 보이게 수정해봐도 슬라이드바 내용 때문에 공간에 제약이 많아서 우측 상단 테마 모드 버튼들이 있는 공간에 위치를 옮겼는데, 옮긴 위치가 좀 더 자연스럽고 적절하다는 생각이 들어서 현재 상태를 유지할 생각이며 추가로 수정할 부분이 남아서 추가 수정한 후 마무리하고 공용 변수 관련 내용을 추가로 처리해줄 생각이다.
