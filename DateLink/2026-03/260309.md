# 오늘 느낀 점

- 그룹 채팅방 드래그 시에 그림자 형태로 보여지는 오버레이 내용이 현재 적용된 테마 모드 색으로 보여지도록 구성한 내용은 문제없이 적용된 것을 확인했고, 사용자 프로필과 사용자 프로필 내의 채팅 입력창 그리고 사용자 세부 내용 또한 현재 적용된 테마 모드 색으로 보여지도록 구성을 했는데, 이 부분에서 적용된 테마 모드 색이 제대로 보이지 않는 문제가 발생해 최상단 바디에 테마 모드가 적용되도록 수정한 결과, 스코프 문제가 해결되어 테마 모드 변경 시에 정상적으로 관련 색이 적용되는 것을 확인해주었다. 그룹 채팅방 추가 버튼이나 사용자 프로필 세부 내용에서 보여지는 메시지 버튼과 친구 버튼 관련 내용이 변경되는 테마 모드에 따라 색이 어떻게 변화될 지 구성하고 테스트해봐야 하는데, 현재 생각한 내용으로는 그룹 채팅방 추가 버튼만 변경된 테마 모드 색이 적용되도록 구성하고 사용자 프로필 세부 내용에 구성된 메시지 버튼과 친구 버튼은 기존 다크 모드 색으로만 유지되도록 하는 방향도 생각중인데, 이 부분은 여러 내용으로 테스트해본 뒤 제일 나아 보이는 UI로 결정할 생각이다.
