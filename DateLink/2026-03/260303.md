# 오늘 느낀 점

- 친구 버튼과 서브 사이드바 내용이 각 테마 모드색에 맞춰 보여지도록 구성한 내용은 자연스럽게 어울리는 것을 확인했고, 블루 모드에서 보여지는 채팅 입력창은 회색이 아닌 파란색 계열로 보여지도록 구성해 테스트해봤지만 조금 어색한 느낌이 들어서 추가 수정이 필요하다고 생각이 된다. 그리고 그룹 채팅방과 채팅 입력창 배경색이 각 테마 모드에 맞는 색으로 보여지도록 수정을 해줄 생각이며, 채팅방에서 이전에 읽은 메시지 위치를 로컬 스토리지에 저장하는 내용이 있는데, 그룹 채팅방 삭제 시에 해당 내용이 함께 삭제되지 않아서 로컬 스토리지에 계속 데이터가 쌓이고 있으므로 이와 관련된 내용을 수정해 그룹 채팅방 삭제 시에 함께 삭제될 수 있도록 구성할 생각이다.
