# 오늘 느낀 점

- 그룹 채팅방 나가기 또는 삭제 시에 로컬 스토리지에 저장된 이전에 읽은 메시지 위치 항목이 삭제되도록 구성을 추가하고, 현재 적용된 테마 모드에 따라 다이렉트 채팅방 패널과 그룹 채팅방 패널 그리고 그룹 채팅방, 그룹 채팅방 서브 사이드바 관련 내용들에서 보여지도록 구성했는데, 해당 내용들 또한 문제없이 보여지고 있기 때문에 추가로 그룹 채팅방을 호버하거나 활성화했을 때 공통적으로 사용되는 하늘색이 아닌 좀 더 테마 모드에 맞는 색으로 보여지도록 수정할 생각이며, 채팅 입력창 또한 현재 적용된 테마 모드에 따라 색이 변화될 수 있도록 수정할 생각이다. 그리고 사용하지 않는 내용이나 주석 처리되어 남겨진 내용들을 확인해 더이상 사용하지 않을 것 같은 내용들은 모두 삭제해 정리해줄 생각이며, 중요하다고 생각되거나 앞으로 다시 사용될 수 있다고 생각된 내용은 남겨놓을 생각이다.
