# 오늘 느낀 점

- 다이렉트 채팅방 패널과 그룹 채팅방 패널에서 펼치기 기능으로 스크롤이 생겨날 경우 최하단의 내용이 약간 잘리는 문제가 발생해 해당 문제를 해결해주었고 다이렉트 채팅방 패널의 스크롤은 기본 UI로 보여지고 있어서 다른 스크롤 내용처럼 자연스럽게 보일 수 있도록 CSS 내용을 추가해주었다. 그리고 공간 관련 CSS 공용 변수를 구성해놓고 정작 적용은 시키지 않은 상태인 것을 확인했기 때문에 해당 내용을 적용시켜야 하며, 공간 관련 CSS 공용 변수에서 좀 더 큰 값은 현재 변수화되지 않은 상태라서 해당 내용은 추가로 확인한 뒤 중복 사용이 많아서 마찬가지로 변수화해줄 생각이고, 전체적으로 한 번 더 점검해 변수화해야 할 내용이 없는지 확인해줄 생각이다.
