# 오늘 느낀 점

- 마진 관련 CSS 내용을 다시 확인한 뒤 변수화하기 애매했던 값들은 CSS 공용 변수로 사용되는 수치로 수정해 CSS 공용 변수를 적용시켜주고 UI를 해칠만한 내용은 굳이 변수화하지 않고 그대로 남겨놓았다. 그리고 온라인, 오프라인 같은 테두리 색이나 취소와 비활성화 버튼 관련 CSS 공용 변수도 추가해 적용해주었고 사용자 프로필 세부 내용에서 내용이 길어질 경우 스크롤이 생겨나지 않는 문제가 발생하고 있었기 때문에 스크롤이 생길 수 있도록 구성하고 스크롤이 내용과 너무 붙은 상태로 보여지기 때문에 약간의 간격을 추가해 좀 더 자연스럽게 보여지도록 구성을 해주었지만 아직 약간 부족하다는 생각이 들어서 추가로 수정해 스크롤이 자연스럽게 보여질 수 있도록 구성할 생각이다. 그리고 테두리 색이나 취소 버튼과 비활성화 버튼처럼 CSS 공용 변수로 수정할 수 있는 내용이 있던 것 처럼 다른 내용도 추가로 있을 수 있으니 더 확인해본 뒤 최대한 CSS 공용 변수로 사용할 수 있도록 구성해줄 생각이다.
